## Phase 1: 短期（〜6ヶ月） — 低難度・高シナジー

現行論文の直接拡張。コードベースの最小改修で実現。

### 論文 A: 反密集区間 + コントラスト密集パターン

| 項目 | 内容 |
|------|------|
| テーマ | 密集区間の対称的拡張（反密集）と 2 レジーム間の構造比較（コントラスト） |
| 新規性 | 密集区間の「消失」を検出する最初の手法 + 位相的変化の分類 |
| 実装量 | 閾値交差方向の反転 + 区間構造比較の検定 |
| データ | 既存の合成データ + 実データ（Dunnhumby / Online Retail） |
| 投稿先 | **DSAA 2026** or **ICDM 2026** |
| 引用関係 | Apriori-Window 原論文 + イベント帰属論文を引用 |
| 工数 | 実装 2 週間 + 実験 2 週間 + 執筆 3 週間 |

### 論文 B: 稀密集パターン

| 項目 | 内容 |
|------|------|
| テーマ | グローバルに稀だが局所的に密集するパターンの検出 |
| 新規性 | 稀パターンマイニング × 密集区間の初の統合 |
| 技術的貢献 | 2 段階マイニング（局所密集条件による候補回復）+ 反単調枝刈りの再定義 |
| 投稿先 | **KDD 2026** or **SDM 2026** |
| 工数 | 実装 3 週間 + 実験 3 週間 + 執筆 3 週間 |

## Phase 2: 中期（6〜12ヶ月） — 中難度・高インパクト

### 論文 C: 多次元密集領域マイニング

| 項目 | 内容 |
|------|------|
| テーマ | 1D→nD 拡張。サポート曲面とそのレベルセットとしての密集領域 |
| 新規性 | 密集領域包含定理、次元分解可能性条件、スウィープ面アルゴリズム |
| 実装量 | 2D プロトタイプ（Python）→ 効率的アルゴリズム（Rust） |
| データ | 合成データ（植え込み密集領域）+ Dunnhumby（店舗位置付き） or 疫学データ |
| 投稿先 | **KDD 2027** or **ICDM 2026** |
| 工数 | 定式化 3 週間 + 実装 4 週間 + 実験 3 週間 + 執筆 4 週間 |

### 論文 D: 因果帰属への昇格

| 項目 | 内容 |
|------|------|
| テーマ | 統計的関連（Phase 2）から合成コントロールによる因果効果推定へ |
| 新規性 | パターンサポート時系列に対する合成コントロール法の初適用 |
| 技術的貢献 | アイテム非共有パターンをドナープールとして活用 |
| 投稿先 | **KDD 2027** or **AAAI 2027** |
| 工数 | 方法論設計 4 週間 + 実装 3 週間 + 実験 3 週間 + 執筆 4 週間 |

### 論文 E: Top-k + 適応的窓幅

| 項目 | 内容 |
|------|------|
| テーマ | θ と W の両方を不要化。分枝限定 + 多スケールスイープ |
| 新規性 | パラメータフリーな密集区間マイニング |
| 投稿先 | **VLDB 2027** or **ICDM 2027** |
| 工数 | 理論 4 週間 + 実装 4 週間 + 実験 3 週間 + 執筆 4 週間 |
