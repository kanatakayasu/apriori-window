# ML/AI との統合

密集パターンマイニングと現代的 ML/AI 技術のシナジーによる研究拡張。

---

## 1. Attention 帰属

**新規性**: High | **難度**: Medium | **投稿先**: KDD / CIKM

### 概要
手作りの帰属スコア `A = prox × dir × mag` を学習型 Cross-Attention に置換。

### アプローチ
- 変化点をベクトル化: `[τ_norm, Δ, direction, pattern_emb]`
- イベントをベクトル化: `[t_start, t_end, duration, event_type_emb]`
- Multi-head cross-attention: queries=変化点, keys/values=イベント
- Attention weight = soft 帰属スコア
- 合成データ（既知 GT）で訓練、置換検定はそのまま維持

### 利点
- 指数減衰を仮定しない非線形時間カーネルの学習
- 遅延効果、先行購買などの複雑な時間関係の捕捉
- パターンタイプとイベントタイプの交互作用の発見

## 2. トランザクション系列基盤モデル

**新規性**: High | **難度**: Medium-Low | **投稿先**: KDD / NeurIPS

### 概要
大規模トランザクション系列で Transformer を事前学習（masked-transaction prediction）。
密集区間検出やサポート予測にファインチューニング。

### 関連研究
TREASURE (arXiv 2511.19693) はトランザクション基盤モデルだが、
不正検出・推薦が対象。密集パターン検出への応用は未探索。

### 利点
- 速度: ウィンドウあたり O(1) 推論（厳密計算の O(n log n) に対して）
- 転移: ドメイン A で事前学習 → ドメイン B に少量ラベルで適応
- ただし近似的: 高速フィルタとして使い、フラグされた窓のみ厳密検証

## 3. 密集パターン特徴量化

**新規性**: High | **難度**: High (実装容易) | **投稿先**: KDD / WWW

### 概要
Apriori-Window の出力を多チャネル二値時系列に変換し、
下流タスク（需要予測、異常検出、離脱予測）の特徴量として利用。

```
B[t, k] = 1  if パターン k が時刻 t で密集
         0  otherwise
→ 形状 (T, K) の二値行列
→ 生特徴量と連結して Temporal CNN / Transformer に入力
```

### 利点
- 密集パターンは事前計算済みの圧縮された解釈可能な特徴量
- ニューラルモデルが共起構造をゼロから再発見する必要なし
- アブレーション: (生のみ) vs (生+密集) で密集特徴量の価値を定量化

## 4. LLM パターン解釈

**新規性**: Medium-High | **難度**: High (実装容易) | **投稿先**: AAAI / CHI

### 概要
マイニング結果を LLM に入力し、自然言語での説明・仮説生成を行う。

### 入力
```json
{
  "pattern": ["bread", "butter", "jam"],
  "dense_intervals": [[100, 150], [300, 320]],
  "attributed_events": [{"name": "Holiday Sale", "score": 0.85}],
  "p_value": 0.003
}
```

### 出力
「パターン {パン, バター, ジャム} はホリデーセール期間中に密集しました。
これら 3 品目は全て朝食関連商品であり、帰省に伴う家庭内消費の増加が考えられます。」

### 利点
- ドメイン知識とマイニング結果の橋渡し
- アナリストが CSV テーブルではなくナラティブを受け取れる

## 5. コントラスティブ学習によるパターン埋め込み

**新規性**: Medium-High | **難度**: Medium | **投稿先**: CIKM / ICDM

### 概要
サポート時系列プロファイルが類似するパターンを近い埋め込み空間に配置。

### 応用
- パターンクラスタリング: 同じイベントに反応するパターン群の発見
- 重複排除の改善: 現行のルールベース（Union-Find）→ 埋め込みベース
- ドメイン間転移: 埋め込み空間の対応関係の学習

## 8. ニューラルサポート推定

**新規性**: Medium | **難度**: High (実装容易) | **投稿先**: VLDB / CIKM

### 概要

予測サポート > θ - margin の窓のみ厳密計算 → 10-100x 高速化。

## 11. パターン間因果ネットワーク

**新規性**: High | **難度**: Medium | **投稿先**: KDD / AAAI

### 概要
パターンサポート時系列間に Granger 因果 / PCMCI を適用し、
パターン間の因果関係を発見。

### 例
「{傘, レインコート} が密集 → 5日後に {風邪薬, ティッシュ} が密集」
→ 天候→購買チェーンの発見
