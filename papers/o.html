イベント帰属の学習

密集パターンを10次元特徴ベクトル（DPFV）に変換し、マルチヘッド交差注意機構で パターン-イベント間の複雑な関係を捉えるCAAMモデルを提案。 統計的手法の一般化能力の限界を深層学習で克服することを目指す。

1. はじめに

Paper A で提案したイベント帰属パイプラインは、円形シフト置換検定とBH FDR補正に基づく統計的フレームワークであり、合成データにおいてF1 = 1.0を達成した。しかしながら、この統計的アプローチには本質的な限界が存在する。新しいイベント種別やパターン構造に対して、手動で設計された帰属スコア（近接度、方向整合性、変化量の積）が最適であるとは限らない。

深層学習、とりわけ注意機構（Attention mechanism）は、入力特徴間の複雑な関係を自動的に学習する能力を持つ。自然言語処理（Vaswani et al., 2017）やコンピュータビジョン（Dosovitskiy et al., 2021）において、Transformer アーキテクチャは従来の手法を大幅に凌駕してきた。

核心的な問い

密集パターンのサポート変動特性とイベント特性を特徴ベクトルとして表現し、交差注意機構（Cross-Attention）によって帰属スコアを学習することで、統計的手法の一般化能力を超えられるか？

本稿では、この問いに答えるため、CAAM（Cross-Attention Attribution Model）を提案する。CAAMは密集パターンを10次元のDense Pattern Feature Vector（DPFV）に変換し、イベントのEvent Feature Vector（EFV）との間でマルチヘッド交差注意を適用することで、スカラー帰属スコアを生成する。

本研究の貢献

  1. DPFV（Dense Pattern Feature Vector）: 密集パターンの時間的・統計的特性を10次元ベクトルに圧縮する特徴量設計。パターンサイズ、密集区間数、サポート統計量、変化点情報、エントロピーを統合。
  2. CAAMモデル: DPFVとEFVの間にマルチヘッド交差注意（h=4, dmodel=64）を適用し、Sigmoid出力で[0,1]帰属スコアを生成するエンドツーエンド学習アーキテクチャ。
  3. 統計的手法との定量比較: 合成データにおけるCAAMのAUC 0.678は、Paper Aの統計的手法（AUC 0.928）に大幅に劣ることを実証し、現段階での深層学習アプローチの限界を明示。

2. 関連研究

2.1 Attention機構とTransformer

Transformer（Vaswani et al., 2017）は自己注意機構（Self-Attention）に基づくアーキテクチャであり、系列データの長距離依存関係を効率的にモデル化する。BERT（Devlin et al., 2019）は双方向エンコーダ表現を学習し、多数のNLPタスクでSOTAを達成した。Cross-Attentionはクエリとキー・バリューが異なるソースに由来する注意機構であり、機械翻訳のデコーダや視覚言語モデル（CLIP, Radford et al., 2021）で広く使用される。

2.2 時系列データへのAttention適用

TFT（Temporal Fusion Transformer, Lim et al., 2021）は時系列予測に特化したAttentionアーキテクチャである。Informer（Zhou et al., 2021）はProbSparse Attentionにより長系列予測を効率化した。PatchTST（Nie et al., 2023）はパッチベースの時系列Transformerを提案した。これらは予測タスクに焦点を当てており、パターン-イベント間の帰属スコアリングへの適用は未開拓である。

2.3 パターンの特徴量設計

頻出パターンの特徴量設計に関する研究は限定的である。Pattern embedding（Ying et al., 2018）はグラフニューラルネットワークを用いてパターンを埋め込み空間に射影する。SubGNN（Alsentzer et al., 2020）はサブグラフの位置・構造的特徴を学習する。しかし、密集パターンの時間的変動特性を特徴量として設計した研究は存在しない。本研究のDPFVは、密集区間マイニングの出力を直接利用する初の特徴量設計である。

2.4 Neural Attribution

ニューラルネットワークの帰属分析（Attribution analysis）は、予測結果に対する入力特徴の寄与を定量化する技術である。Integrated Gradients（Sundararajan et al., 2017）、SHAP（Lundberg & Lee, 2017）、Attention-based attribution（Jain & Wallace, 2019）が代表的手法である。これらはモデル内部の帰属であり、本研究が扱う外部イベントへの帰属とは問題設定が異なるが、注意重みの解釈可能性に関する知見は参考になる。

3. 問題定義

3.1 Dense Pattern Feature Vector (DPFV)

定義 1: DPFV

密集パターン P とその密集区間集合 I(P) に対して、Dense Pattern Feature Vector DPFV(P) を以下の10次元ベクトルとして定義する。

DPFV(P) = [f1, f2, ..., f10] ∈ R10
次元特徴量説明
f1パターンサイズ|P|: パターンに含まれるアイテム数
f2密集区間数|I(P)|: 検出された密集区間の総数
f3最長密集区間maxi(ei - si): 最も長い密集区間の長さ
f4総密集期間Σi(ei - si): 全密集区間の長さの合計
f5平均サポートs̄P: サポート時系列の平均値
f6サポート標準偏差σ(sP): サポート時系列の標準偏差
f7変化点数閾値交差による変化点の総数
f8最大変化量maxτ magnitude(τ): 最大のレベルシフト量
f9時間中央値密集区間の時間的中心の中央値
f10エントロピーサポート時系列の正規化エントロピー

3.2 Event Feature Vector (EFV)

定義 2: Event Feature Vector (EFV)

外部イベント e = (id, name, ts, te) に対して、Event Feature Vector EFV(e) を以下で定義する。

EFV(e) = [ts, te, te - ts, emb(type(e))] ∈ Rdefv

ここで emb(type(e)) はイベント種別の学習可能な埋め込みベクトル（demb=16）である。

開始時刻 ts

正規化されたイベント開始時刻

終了時刻 te

正規化されたイベント終了時刻

期間 Δt

te - ts: イベントの持続時間

3.3 帰属スコア

定義 3: 帰属スコア

パターン P とイベント e の帰属スコア a(P, e) を以下で定義する。

a(P, e) = CAAM(DPFV(P), EFV(e)) ∈ [0, 1]

CAAMはマルチヘッド交差注意に基づくニューラルネットワークであり、エンドツーエンドで教師あり学習される。

4. 提案手法: CAAM (Cross-Attention Attribution Model)

CAAMは4つのステップから構成される。Phase 1（Apriori-window）の出力とイベントメタデータを入力とし、学習済みモデルによりスカラー帰属スコアを出力する。

1

Step 1: DPFV抽出

各密集パターン P に対して、Phase 1 の出力（密集区間リスト、サポート時系列）から10次元の特徴ベクトルを計算する。パターンサイズ、密集区間の統計量、サポート時系列の統計量、変化点情報、エントロピーを統合する。全特徴量は標準化（z-score正規化）される。

2

Step 2: EFV抽出

各イベント e のメタデータ（開始時刻、終了時刻、種別）から特徴ベクトルを構成する。時刻情報は [0,1] に正規化し、イベント種別は学習可能な埋め込み層（demb=16）を通じてベクトル化する。EFVの次元は defv = 3 + 16 = 19 となる。

3

Step 3: マルチヘッド交差注意

DPFVをクエリ Q、EFVをキー K およびバリュー V として、マルチヘッド交差注意を計算する。

Q = WQ · DPFV(P),  K = WK · EFV(e),  V = WV · EFV(e)

Attention(Q, K, V) = softmax(QKT / √dk) · V

MultiHead = Concat(head1, ..., headh) · WO

h=4ヘッド、dmodel=64、dk=dv=16 の構成を使用する。出力はフィードフォワード層（FFN: 64→32→1）に入力される。

4

Step 4: Sigmoid出力

FFN の出力にSigmoid関数を適用し、[0,1] の帰属スコアを生成する。

a(P, e) = σ(FFN(MultiHead(Q, K, V)))

損失関数はBinary Cross-Entropy（BCE）を使用し、ラベル付き帰属データで教師あり学習する。

学習データの生成

合成データに対してPaper Aの統計的手法でラベルを生成し、CAAMの教師データとする。具体的には、統計的手法が有意と判定した(P, e)ペアを正例、それ以外を負例とする。Train/Val/Test = 70/15/15 の分割を使用する。

考察

CAAMはMLPおよびRFを上回る（AUC +0.086, +0.077）が、Paper Aの統計的手法（AUC 0.928）には大幅に劣る（ΔAUC = -0.250）。交差注意による特徴間相互作用の学習は一定の効果を示すものの、合成データの規模（N=5,000, 3イベント）では統計的手法の精度を超えるのに十分な複雑性が存在しない。

6. 結論

本稿では、密集パターンの10次元特徴ベクトル（DPFV）とイベント特徴ベクトル（EFV）に対してマルチヘッド交差注意機構を適用するCAAM（Cross-Attention Attribution Model）を提案した。

主要な実験結果

  • CAAMのAUC 0.678はMLP（0.592）およびRF（0.601）を上回り、交差注意による特徴間相互作用の学習効果を確認
  • 注意重み分析により、時間的特徴（変化点数、時間中央値、最大変化量）が帰属において最重要であることを発見
  • しかし、統計的手法（AUC 0.928）にはAUCで0.250劣る結果となった
  • 注意重みが一様分布に近く（最大/最小比 = 3.07）、判別的な表現学習が不十分

制約と限界

  • AUC 0.678 は統計的手法に大幅に劣り、実用的な帰属性能には到達していない
  • 合成データのみでの評価であり、実データでの検証が不足
  • 注意重みの一様分布は、モデルが有意義な特徴間関係を学習できていないことを示唆
  • ラベル付き帰属データの生成自体が統計的手法に依存しており、循環的な問題がある
  • NumPyベースの簡易実装であり、PyTorchによる本格的なエンドツーエンド学習が必要

今後の課題

  1. PyTorchによるエンドツーエンド実装と勾配ベースの最適化
  2. 大規模データセット（N ≥ 100K）での学習と評価
  3. 時系列全体を入力とするTransformerエンコーダの導入（Paper Q への接続）
  4. 半教師あり学習によるラベル依存の軽減
  5. 注意重みの正則化（スパース性促進）による解釈可能性の改善

本研究は概念実証の段階にあり、統計的手法を上回る性能を達成するまでは保留とする。PyTorch実装と大規模データでの実験が今後の優先課題である。
