密集区間予測：
記述的から予測的パターンマイニングへ

密集区間マイニングの出力を点過程・生存分析と接続し、「次にいつ密集が再発するか」「どれくらい持続するか」を予測するHawkes-Denseフレームワークを提案。

Wave 4 Hawkes過程 Weibull生存分析 Poisson退化保証 相対誤差 <2% KDD/SDM

1. はじめに

頻出パターンマイニングとその時間的拡張（密集区間マイニングを含む）は、本質的に記述的な手法である。「どのパターンがいつ頻出であったか」を事後的に報告するが、将来の密集区間の発生を予測する機能を持たない。

しかし実応用では、予測的な知見がしばしばより有用である。小売業では「次にこの商品組み合わせの販売バーストがいつ起きるか」を知ることで在庫最適化が可能になり、製造業では「次にこのアラームパターンが再発するか」を予測することで予防保全が実現する。

核心的な問い

密集区間マイニングが出力する密集区間の発生時点列と持続時間列は、本質的に点過程と生存時間として解釈できる。この構造的対応を利用すれば、記述的マイニングの結果を予測モデルの入力として活用できる。

本研究の貢献

  1. DIOP定式化: 密集区間の開始時点列を点過程として定式化し、Inter-Dense Interval Time (IDIT) を定義
  2. Hawkes-Dense モデル: 自己励起Hawkes過程で次回密集区間発生時刻を予測。自己励起がない場合はPoisson過程に適切退化する健全性を証明
  3. Weibull持続時間モデル: Weibull分布による密集区間の持続時間予測

2. 関連研究

2.1 密集区間マイニング

Apriori-windowは固定幅スライディングウィンドウ上で局所サポートを評価し、閾値以上が連続する区間を密集区間として出力する。本研究はこの出力を入力として予測モデルに接続する。

2.2 Hawkes過程と点過程

Hawkes過程（Hawkes 1971）は自己励起型の点過程であり、過去のイベント発生が将来の発生確率を一時的に増加させる。地震学（余震モデリング）、金融（注文フロー）、ソーシャルメディア（情報拡散）で広く応用されている。しかし、頻出パターンの密集区間発生をHawkes過程でモデル化した研究は存在しない。

2.3 生存分析

Kaplan-Meier推定、Cox比例ハザードモデル（Cox 1972）、パラメトリック生存モデル（Weibull、Log-normal）は、イベントまでの時間をモデル化する基盤的手法である。製造業での故障までの時間（MTTF）予測に広く用いられている。

2.4 時系列予測

ARIMA、Prophet、深層学習ベース（LSTM、Transformer）の時系列予測は連続値の予測に適するが、密集区間の「発生/非発生」というイベント構造を直接扱わない。本研究はイベントの発生時点を点過程として扱う。

3. 問題定義

定義 1: Dense Interval Onset Process (DIOP)

パターン P の密集区間列を [s1,e1], ..., [sK,eK] とする。P の DIOP は開始時点列 {s1, ..., sK} として定義される。

定義 2: Inter-Dense Interval Time (IDIT)

連続する密集区間間の待ち時間: IDITk = sk+1 - (sk + durk)。ここで durk = ek - sk + 1。

定義 3: 予測問題

過去の DIOP と持続時間列が与えられたとき、(a) 次の密集区間開始時刻 sK+1 と (b) 持続時間 durK+1 の期待値を予測する。

Hawkes強度関数

λ(t) = μ + Σsi<t α · exp(-β(t - si))

μ > 0: ベースライン強度、α ≥ 0: 励起係数、β > 0: 減衰率。安定条件: α/β < 1。

4. 提案手法: Hawkes-Dense フレームワーク

1

DIOP / IDIT 系列の抽出

各パターンの密集区間列から開始時点列（DIOP）と待ち時間列（IDIT）を抽出。最低 K ≥ 5 の密集区間を持つパターンのみを対象とする。

2

Hawkes-Dense モデルによる発生時刻予測

DIOPに対してHawkes過程をフィット（MLE推定）し、条件付き強度関数から次回発生時刻の期待値を計算。

E[sK+1 | Ht] = t + 1/λ(t)
3

Weibull 生存モデルによる持続時間予測

持続時間列にWeibull分布をフィット。k > 1 は摩耗型、k < 1 は早期故障型、k = 1 は指数分布（メモリレス）に退化。

f(d; k, λ) = (k/λ)(d/λ)k-1 exp(-(d/λ)k)

4.4 Poisson退化の健全性

命題: Poisson退化

α → 0 のとき Hawkes 強度関数は λ(t) = μ（定数）に退化し、DIOPは rate μ の定常Poisson過程となる。自己励起効果が存在しないデータに対してモデルは過適合せず、最も単純なベースラインに適切に退化する。

6. 結論

主要な結果

  • Hawkes過程パラメータを相対誤差 <5% で回復
  • 次回密集区間発生時刻の予測で相対誤差 <2%
  • 自己励起がない場合にPoisson過程へ適切退化（過適合なし）
  • Weibull分布で持続時間を予測し、exponentialを有意に上回る

制約と今後の課題

  • 合成データのみの評価; 実データ（小売、製造）での外的妥当性検証が必要
  • べき乗則カーネルや多変量Hawkesは未検討
  • K < 5 の少数密集パターンには適用できない
  • ニューラルHawkes過程との比較が今後の課題
