1. はじめに

頻出パターンマイニング (Agrawal 1994, Han 2000, Fournier-Viger 2017) は、トランザクションデータ中に繰り返し出現するアイテムセットを発見する基盤的技術であり、小売分析、Webマイニング、バイオインフォマティクスなど幅広い分野で応用されている。Apriori、FP-Growth、ECLATといった古典的手法はグローバル頻出パターンを効率的に列挙し、スライディングウィンドウ型手法は特定時間区間内で局所的に密集するパターンを検出する。

しかし、これらの手法は「どの」パターンが「いつ」頻出であるかを同定するのみであり、「なぜ」パターンの出現頻度が変動したかを説明する機能を持たない。

核心的な問い

実際の応用場面では、販促キャンペーン、祝日、規制変更、システム更新といった外部イベントがパターンサポートの時間的変動を駆動していることが多い。「どのイベントがどのパターンの変動を引き起こしたか」を特定できれば、キャンペーン効果の定量化、副作用の検出、システム異常の診断など、意思決定に直結する知見が得られる。

このように帰属タスクは実用上重要であるにもかかわらず、頻出パターンのサポート変動を外部イベントに結びつける統計フレームワークは、我々の知る限り存在しない。

本研究の貢献

本稿では、頻出パターンマイニングと外部イベント帰属を橋渡しする統計フレームワーク「イベント帰属パイプライン」を提案する。貢献は以下の3点である。

  1. 5ステップパイプライン: 密集区間からの変化点抽出、変化量の評価、変化点とイベント間の帰属スコアリング、円形シフト置換検定によるグローバル Benjamini-Hochberg FDR補正、およびアイテム重複排除からなる統合フレームワーク。
  2. アイテム重複排除: 真に影響を受けたパターンとアイテムを共有する副次パターンが相関した偽陽性を生む問題（副次パターン問題）を、イベントごとの Union-Find クラスタリングにより解決する新しい後処理技法。
  3. 包括的な実験評価: (i) 合成データにおける完全検出（F1 = 1.0）、(ii) パラメータ選択に対する頑健性、(iii) 100万トランザクションまでの線形スケーラビリティ、(iv) 実データにおける偽陽性96%削減を実証する。

2. 関連研究

2.1 頻出パターンマイニング

頻出パターンマイニングは、トランザクションデータの一定割合以上に出現するアイテムセットを発見する技術であり、包括的なサーベイが複数存在する。Apriori (Agrawal 1994) は反単調性に基づく候補生成・枝刈りフレームワークを確立した。FP-Growth (Han 2000) はFP-treeにより候補生成を回避し高速化を実現した。ECLAT (Zaki 2000) は縦型（tidset）表現と集合積により効率的なサポート計算を行う。LCM (Uno 2004) は頻出・閉・極大アイテムセットの最適列挙を達成した。これらの手法はグローバル頻度を対象とし、時間的な変動をモデル化しない。

2.2 時間的・局所的パターンマイニング

頻出パターンの時間的側面を扱う拡張研究は多岐にわたる。LPFIM (Mahanta 2005) は時間区間内の局所頻出集合を対象とするが、gapベースの閾値に依存する。PFPM (Tanbeer 2009) および PPFPM (Kiran 2017) は周期性制約を導入した。LPPM (Fournier-Viger 2021) は局所性と周期性の基準を統合する。Apriori-windowは固定幅スライディングウィンドウ上で局所サポートを直接評価し、gapパラメータに依存しない密集パターン抽出を実現した。

これらの手法はいずれもパターンがいつ頻出であるかを検出するが、頻度が変動する原因はモデル化しない。本研究はこれらを補完し、検出されたサポート変動を外部イベントに帰属させる。

2.3 パターン変化のモニタリングと診断

パターンの時間的変動を追跡する研究群が存在する。Agrawal と Psaila (1995) は、ルール統計量の履歴に対する形状クエリによりトレンドを検出する「アクティブデータマイニング」を提案した。Chakrabarti ら (1998) は記述長に基づくサプライズ指標を用いて、サポートが予想外に変動するアイテムセットを同定した。DEMON (Ganti 2001) は進化するデータベースのモデル維持を効率化し、速度密度推定により変化の種類を診断する。StreamKrimp (van Leeuwen 2008) はMDLに基づく圧縮でストリーム中のパターン分布変化を検出する。

これらの手法は変化の検出と特徴づけを行うが、検出された変化を外部イベントに帰属させる統計フレームワークは持たない。

2.4 コントラスト・エマージングパターンマイニング

エマージングパターン (Dong 1999) は、あるデータセットから別のデータセットへのサポートの有意な増加を捉える。コントラストセットマイニング (Bay 2001) は、グループ間の判別的特徴を発見する。Komiyama ら (2017) は多重検定補正を伴う統計的エマージングパターンマイニングを提案した。これらの手法は何が変わったかを検出するが、なぜ変わったかは説明しない。

2.5 変化点検出

CUSUM (Page 1954) は逐次過程の平均シフトを検出する。これらの手法は個別時系列に適用されるが、数千のパターンサポート時系列を同時分析する多重検定の負荷には対応せず、検出された変化をイベントに結びつける機能も持たない。

2.6 イベント帰属・介入分析と因果推論

Box-Tiao介入分析 (1975) はARIMAモデルにステップ/パルス指示変数を組み込んで既知の介入効果を推定する。中断時系列回帰（ITS）は介入前後の水準変化と傾斜変化をセグメント回帰で評価する。イベント・スタディ法 (MacKinlay 1997) は企業イベントの株価への影響を異常リターン分析により定量化する。CausalImpact (Brodersen 2015) はベイズ構造時系列モデルにより介入の因果効果を推定する。

これらのアプローチは典型的には単一または少数のアウトカム変数を対象とし、数千のパターンサポート時系列を複数のイベントに対して同時に検定する設定には直接適用できない。

2.7 統計的パターンマイニングと多重検定補正

Webb (2007) は自己充足アイテムセットの概念を導入した。LAMP (Terada 2013) は検定可能なパターン数に基づくBonferroni因子の最小化を実現した。Llinares-Lopez ら (2015) はTaroneの検定可能性基準とWestfall-Young置換を組み合わせ、FWER制御下での高速な有意パターンマイニングを実現した。本パイプラインは、円形シフト置換検定とグローバルBH補正を組み合わせ、分布自由な検定と適切な多重検定制御を両立させる。

3. 問題定義

3.1 記法

I = {i1, i2, ..., im} を有限のアイテム全集合、D = [d1, d2, ..., dN] を時刻順のトランザクション列とする。各トランザクション dt は I の部分集合である。

定義 1: サポート時系列

パターン P、ウィンドウサイズ W、トランザクション列 D（長さ N）に対して、P のサポート時系列 sP = [sP(0), sP(1), ..., sP(N-W)] を以下で定義する。

sP(t) = |{dj ∈ D | t ≤ j < t + W, P ⊆ dj}|

これは位置 t を始点とするウィンドウ [t, t+W) 内で P を含むトランザクション数である。

定義 2: 外部イベント

外部イベント e = (id, name, ts, te) は、一意な識別子、名称、およびイベントが有効な期間を表すタイムスタンプ区間 [ts, te] から成る。

定義 3: 変化点（閾値交差）

サポート時系列 sP に対する変化点として、閾値 θ に関する密状態／非密状態の切替点を扱う。

上昇: sP(τ) ≥ θ ∧ (τ = 0 ∨ sP(τ-1) < θ)
下降: sP(τ) < θ ∧ τ > 0 ∧ sP(τ-1) ≥ θ

変化量は交差前後の局所平均差: magnitude(τ) = |s̄+(τ) - s̄-(τ)|

3.2 イベント帰属問題

定義 4: イベント帰属問題

頻出パターン集合 F とそのサポート時系列、外部イベント集合 E、有意水準 α が与えられたとき、イベント帰属問題とは、sP の閾値交差変化点がイベント e の時間窓と統計的に有意に関連する全ての組 (P, e) を、偽発見率（FDR）を水準 α 以下に制御しながら同定することである。

この問題の主要な課題は以下の3点である:

規模

|F| 個のパターンと |E| 個のイベントに対して仮説数が O(|F| × |E|) となり、厳格な多重検定補正が必要。

副次パターン

イベントがアイテム i の頻度を増加させると、i を含む全てのパターンのサポートが相関して変動し、偽陽性が大量に発生。

時間的混同

無関係なイベントの時間窓が重複する場合、帰属が曖昧になる。

4. 提案手法: イベント帰属パイプライン

Phase 1（Apriori-window）で得られたパターン集合 F とその密集区間、および外部イベント集合 E を入力として、統計的に有意なパターン--イベント帰属の集合を出力する。

1

密集区間からの変化点抽出

各密集区間 [si, ei] から上昇変化点 si と下降変化点 ei+1 を抽出。パターン出現リスト上の二分探索により O(log |TP|) で局所サポートを計算する。

2

変化量の評価

交差前後の窓平均の差（レベルシフト量）として変化量を計算。瞬時差分と比較して、一過性の変動ではなく持続的な変化を捉える。サポート振幅フィルタと変化量フィルタにより仮説空間を削減する。

magnitude(τ) = |s̄+(τ) - s̄-(τ)|
3

帰属スコアリング

パターン P の変化点 τ と各イベント e に対して帰属スコアを計算。

A(P, τ, e) = prox(τ, e) · dir(τ, e) · mag(τ)

近接度 (prox)

指数カーネル: exp(-d(τ,e)/σ)

方向整合性 (dir)

イベント開始時上昇=1.0、終了時下降=1.0、その他=0.5

変化量 (mag)

レベルシフト量

4

置換検定とグローバルFDR補正

円形シフト置換検定: ランダムオフセットで全イベント時刻を一斉にずらし、帰無分布を生成。

praw(P, e) = (1 + |{b : Aperm(b)(P,e) ≥ Aobs(P,e)}|) / (B + 1)

Benjamini-Hochberg手続き: 全パターン・全イベントにわたりグローバルにFDRを制御。PRDS条件の充足により、アイテムを共有するパターン間の正の相関下でもFDR制御が保証される。

5

アイテム重複排除

副次パターン問題: イベントがアイテム i をブーストすると、i を含む全パターンが有意と判定される。Union-Find クラスタリングによるアイテム共有グラフの連結成分分析で、各成分内のスコア最大パターンのみを残す。

命題: 重複排除の正当性

ブースト確率 β がベースライン出現確率 pbase を超えれば、真のパターン P* のサポート増分 (∝ β2) は副次パターンの増分 (∝ β · pbase) を上回り、P* がスコア最大として保持される。

6. 結論

本稿では、頻出パターンのサポート時系列における変動を外部イベントに統計的に帰属させるイベント帰属パイプラインを提案した。本手法は以下の5ステップから成る:

  1. Phase 1（Apriori-window）で得られた密集区間からの変化点抽出
  2. 交差前後のレベル差による変化量評価
  3. 時間的近接度・方向整合性・変化量に基づく帰属スコアリング
  4. 円形シフト置換検定とグローバル Benjamini-Hochberg FDR 補正
  5. Union-Find クラスタリングによるアイテム重複排除

主要な実験結果

  • 全信号強度（ブースト係数 0.1-0.8）で真のパターン--イベント帰属を完全に回復（F1 = 1.0）
  • ウィンドウサイズ、減衰パラメータ、置換回数に対して頑健
  • 100万トランザクションまで線形スケール（約6.6分）
  • 実データで偽陽性を96%削減
  • Dunnhumby実キャンペーンデータで全30キャンペーンをカバー

制約と限界

  • 本手法は時間的共起の検定であり、因果効果を立証するものではない
  • 複数イベントが時間的に重複する場合、帰属が曖昧になりうる
  • 高ベースラインサポートのパターンでは検出力が低下する
  • W = 200 では検出が完全に失敗する
