\begin{abstract}
取引データにおいては，同一時間帯に複数の独立した購買イベント（バスケット）が記録される
場合が多い．従来の頻出パターンマイニング手法は，1 トランザクションを 1 つのアイテムセット
として扱う設計を前提とするため，複数バスケットを束ねる際に異なる購買イベント由来のアイテム
が誤って共起とみなされる「偽共起」が生じる．本稿では，この問題を解決するためにバスケット
構造を明示的に扱う拡張モデルを提案する．提案手法（Phase 1）は，共起の定義をバスケット内
に限定することで，偽共起パターン率（SPR: Spurious Pattern Rate）を原理的にゼロに抑える
ことを保証する．合成データ実験では，バスケット数パラメータ $\lambda_{\text{baskets}}$ の
増加とともに従来法の SPR が単調に増加する一方，提案手法は SPR = 0 を維持することを
実証する．また，反単調性がバスケット内共起においても成立することを証明し，Apriori 枝刈り
による効率的な候補生成が可能であることを示す．
\end{abstract}
