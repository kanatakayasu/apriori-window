LLM による密集アイテムセットパターンの自然言語解釈

密集パターン検出結果を大規模言語モデルで自動解釈するフレームワーク

KDD Demo / AAAI Demo Paper P Python

概要

密集アイテムセットマイニングは、スライディングウィンドウ内で共起頻度が閾値を超えるアイテムの組み合わせが連続する時間区間を特定する手法です。 しかし、検出されたパターンの解釈はドメイン専門家に依存しており、分析のボトルネックとなっています。

本研究では、密集パターンの検出結果をLLMに入力し、自然言語による解釈（要約・時間的特徴・ビジネス仮説）を自動生成するフレームワークを提案します。

パターン特徴量

  • カバー率: 密集区間がデータ全体に占める割合
  • 区間数: 密集区間の個数
  • 最大ギャップ: 連続する密集区間間の最大距離
  • 時間パターン分類: 単一連続 / 断続的 / 高頻度断続

3種類のプロンプト

  • 単一パターン解釈: 1パターンの詳細な自然言語解釈
  • 比較分析: 複数パターンの関連性分析
  • バッチ要約: 全パターンの一括要約
