% ==============================================================
\section{関連研究}\label{sec:related}
% ==============================================================

\subsection{頻出パターンマイニング}

頻出パターンマイニングは，トランザクションデータの一定割合以上に出現するアイテムセットを発見する技術であり，
包括的なサーベイが複数存在する~\cite{fournier2017survey,aggarwal2014frequent}．
Apriori~\cite{agrawal1994fast}は反単調性に基づく候補生成・枝刈りフレームワークを確立した．
FP-Growth~\cite{han2000mining}はFP-treeにより候補生成を回避し高速化を実現した．
ECLAT~\cite{zaki2000scalable}は縦型（tidset）表現と集合積により効率的なサポート計算を行う．
LCM~\cite{uno2004lcm}は頻出・閉・極大アイテムセットの最適列挙を達成した．
これらの手法はグローバル頻度を対象とし，時間的な変動をモデル化しない．

\subsection{時間的・局所的パターンマイニング}

頻出パターンの時間的側面を扱う拡張研究は多岐にわたり，
Segura-Delgado ら~\cite{segura2020temporal}が包括的なサーベイを提供している．
LPFIM~\cite{mahanta2005finding}は時間区間内の局所頻出集合を対象とするが，
gap ベースの閾値に依存する．
PFPM~\cite{tanbeer2009discovering}およびPPFPM~\cite{kiran2017discovering}は
周期性制約を導入した．
LPPM~\cite{fournier2021mining}は局所性と周期性の基準を統合する．
Ho ら~\cite{ho2022efficient}は相互情報量を用いた枝刈りにより
大規模時系列からの時間的パターンマイニングを高速化した．
Apriori-window~\cite{dsaa2025}は固定幅スライディングウィンドウ上で
局所サポートを直接評価し，gapパラメータに依存しない密集パターン抽出を実現した．
ストリームデータに対しては，スライディングウィンドウ上の頻出アイテムセット
マイニング~\cite{chang2005estwin,chi2004moment}が研究されている．
Batal ら~\cite{batal2012mining}は多変量時系列からの
最近時間パターンマイニングによるイベント検出を提案した．

これらの手法はいずれもパターンが\emph{いつ}頻出であるかを検出するが，
頻度が変動する\emph{原因}はモデル化しない．
本研究はこれらを補完し，検出されたサポート変動を外部イベントに帰属させる．

\subsection{パターン変化のモニタリングと診断}

パターンの時間的変動を\emph{追跡}する研究群が存在する．
Agrawal と Psaila~\cite{agrawal1995active}は，
ルール統計量の履歴に対する形状クエリによりトレンドを検出する
「アクティブデータマイニング」を提案した．
Chakrabarti ら~\cite{chakrabarti1998mining}は
記述長に基づくサプライズ指標を用いて，サポートが予想外に変動する
アイテムセットを同定した．
Liu ら~\cite{liu2001fundamental}は
二時点間のルール変化から「基本的変化」（他の変化から導出できないもの）を
抽出する手法を提案し，冗長な変化の除去を試みた．
DEMON~\cite{ganti2001demon}は進化するデータベースのモデル維持を効率化し，
速度密度推定により変化の種類（消散・凝集・シフト）を診断する．
Yoo と Shekhar~\cite{yoo2009spamine}はサポートプロファイル（時間軸上のサポート推移）の
類似性に基づくパターン検索を提案した．
StreamKrimp~\cite{vanleeuwen2008streamkrimp}はMDLに基づく圧縮でストリーム中のパターン分布変化を検出する．
B{\"o}ttcher と Spiliopoulou~\cite{bottcher2011contrast}は
コントラストマイニングと変化マイニングを統合したサーベイを提供している．

これらの手法は変化の\emph{検出}と\emph{特徴づけ}を行うが，
検出された変化を\emph{外部イベント}に帰属させる統計フレームワークは持たない．
本研究はこのギャップを埋め，変化点とイベントの統計的関連を検定する．

\subsection{コントラスト・エマージングパターンマイニング}

データセット間のパターン頻度の\emph{差異}に着目する研究群が存在する．
エマージングパターン~\cite{dong1999discovering}は，あるデータセットから
別のデータセットへのサポートの有意な増加を捉えるパターンである．
コントラストセットマイニング~\cite{bay2001detecting}は，
グループ間の判別的特徴を発見する．
Novak ら~\cite{novak2009supervised}は，コントラストセット，エマージングパターン，
サブグループ発見を「教師あり記述的ルール発見」として統一し，
いずれも\emph{何が}異なるかを記述する手法であることを示した．
Komiyama ら~\cite{komiyama2017statistical}は
多重検定補正を伴う統計的エマージングパターンマイニングを提案した．
Chen ら~\cite{chen2022contrast}がコントラストパターンマイニングの
包括的サーベイを提供している．
Adams ら~\cite{adams2023explainable}はプロセスマイニングにおける概念ドリフトを
Granger因果で説明する試みを行ったが，内部プロセス指標間の因果関係を検定するものであり，
既知の外部イベントカレンダーへの帰属は行わない．
Gupta ら~\cite{gupta2019frequent}はRubin--Neyman因果モデルに基づく
「頻出因果パターンマイニング」を提案したが，
横断的な治療効果推定であり，時系列上のサポート変動と外部イベントの帰属問題とは異なる．

これらの手法は\emph{何が}変わったかを検出するが，
\emph{なぜ}（どのイベントにより）変わったかは説明しない．
本研究はサポート変動の検出に留まらず，統計検定による外部イベントへの帰属まで踏み込む．

\subsection{変化点検出}

変化点検出は時系列データの構造変化を同定する技術である．
CUSUMアルゴリズム~\cite{page1954continuous}は逐次過程の平均シフトを検出する．
包括的なサーベイ~\cite{aminikhanghahi2017survey,truong2020selective}が
パラメトリック・ノンパラメトリック・オンライン手法を網羅している．
これらの手法は個別時系列に適用されるが，
数千のパターンサポート時系列を同時分析する多重検定の負荷には対応せず，
検出された変化をイベントに結びつける機能も持たない．

\subsection{イベント帰属・介入分析と因果推論}

既知の介入が時系列に与える影響を定量化する手法は複数の分野で研究されている．
Box と Tiao~\cite{box1975intervention}は介入分析の基礎を確立し，
ARIMA モデルにステップ/パルス指示変数を組み込んで
既知の介入効果を推定するフレームワークを提案した．
中断時系列回帰（ITS）~\cite{bernal2017interrupted}は
公衆衛生分野で広く用いられる準実験デザインであり，
介入前後の水準変化と傾斜変化をセグメント回帰で評価する．
計量経済学のイベント・スタディ法~\cite{mackinlay1997event}は，
企業イベントの株価への影響を異常リターン分析により定量化する．
合成コントロール法~\cite{abadie2010synthetic}は
コントロール群の加重平均から反事実を構築し，介入効果を推定する．
Brodersen ら~\cite{brodersen2015causalimpact}はベイズ構造時系列モデルにより
介入の因果効果を推定するCausalImpactフレームワークを提案した．
Granger因果~\cite{granger1969investigating}は
時系列間の予測的因果関係を検定する．

分散システムにおける根本原因分析~\cite{chen2004pinpoint,jeyakumar2019explainit,liu2021root}は，
異常を特定のコンポーネントに帰属させる．
Adtributor~\cite{bhagwan2014adtributor}は
KPI変動をデータ次元（国，ブラウザ等）に帰属させるが，
時間的イベントへの帰属ではない．
Donges ら~\cite{donges2016event}は
イベント時系列間の共起の統計的有意性を検定するイベント一致分析を提案した．
プロセスマイニング分野では，Banham ら~\cite{banham2022xpm}が
外部データ（ICU病床数，天候等）をプロセスモデルに統合するxPMフレームワークを提案したが，
プロセスの制御フロー分析であり，アイテムセットのサポート時系列への帰属ではない．

これらのアプローチは典型的には単一または少数のアウトカム変数を対象とし，
数千のパターンサポート時系列を複数のイベントに対して
同時に検定する設定には直接適用できない．
本研究は，我々の知る限り，パターンマイニングの出力（サポート時系列）と
外部イベントの帰属を統合する初の統計フレームワークを提供する．

\subsection{統計的パターンマイニングと多重検定補正}

頻出パターンマイニングに統計的有意性検定を導入する研究が近年活発である．
Webb~\cite{webb2007discovering}は自己充足アイテムセットの概念を導入し，
サブセット・スーパーセットの頻度から説明できないパターンのみを有意とする
枠組みを提案した．
Webb~\cite{webb2011filtered}はさらに「生産的」ルールの概念により
冗長な有意パターンのフィルタリングを提唱した．
H{\"a}m{\"a}l{\"a}inen と Webb~\cite{hamalainen2019tutorial}はパターン発見における統計的健全性の
チュートリアルを提供している．
Terada ら~\cite{terada2013statistical}はLAMP（Limitless Arity Multiple-testing Procedure）を提案し，
検定可能なパターン数に基づくBonferroni因子の最小化を実現した．
Llinares-L{\'o}pez ら~\cite{llinares2015fast}はTaroneの検定可能性基準と
Westfall--Young置換~\cite{westfall1993resampling}を組み合わせ，
FWER制御下での高速な有意パターンマイニングを実現した．
Pellegrina と Vandin~\cite{pellegrina2020significant}は同様のフレームワークを拡張した．
Lijffijt ら~\cite{lijffijt2014statistical}は
最大エントロピーモデルの逐次更新により非冗長な有意パターン集合を選択する手法を提案した．

これらの研究はパターンの有意性を二群間の頻度差として検定するが，
本研究ではパターンサポートの\emph{時間的変動}と\emph{外部イベント}の
関連を検定する点が異なる．
本パイプラインは，サロゲートデータ法~\cite{theiler1992testing}の考え方に基づく
円形シフト置換検定~\cite{davison1997bootstrap,yuan2024rigorous,haiminen2008pairwise}と
全パターン・全イベントにわたるグローバルBH補正~\cite{benjamini1995controlling}を組み合わせ，
分布自由な検定と適切な多重検定制御を両立させる．
BH手続きの依存仮説への拡張~\cite{benjamini2001control}により，
アイテムを共有するパターン間の正の相関下でもFDR制御が保証される．
Bonferroni補正~\cite{dunn1961multiple}は族単位誤り率（FWER）を制御するが保守的であるため，
本研究ではFDRを制御するBH手続きを採用した．
