\section{結論}\label{sec:conclusion}
% ==============================================================

本稿では，頻出パターンのサポート変動を外部イベントに帰属させる
\textbf{イベント帰属問題}を新たに形式的に定義し，
FDR制御付き多重検定問題としての3つの構造的課題
（仮説空間の規模，副次パターンによる相関偽陽性，時間的混同）を同定した．
本研究の主貢献は新しいマイニングアルゴリズムの提案ではなく，
この未開拓な問題の定式化と初の統計的解法フレームワークの構築にある．
個々の構成要素（変化点検出，置換検定，BH補正，Union-Find）は
それぞれ確立された技術であるが，
これらをパターンサポートの変動と外部イベントの帰属という
統合的な問題に適用する枠組みは存在しなかった．
解法として5ステップのパイプラインを構成し，
帰属スコアはランキング用指標に留め，
最終判定を置換検定に委ねる二層構造により，
ヒューリスティックな閾値選択への依存を軽減した．

実験により以下の知見を得た:
\begin{itemize}
\item Zipf分布を含む多様なデータ条件において高い帰属精度を達成し，
帰無条件下では20 seedで平均偽発見率0.05（$\leq \alpha = 0.10$）を達成し，FDR制御を実証した（EX1）．
\item アブレーション分析により，proximity と magnitude の相補性，
およびグローバルBH補正と重複排除の段階的な偽陽性削減効果を実証した（EX2）．
\item 関連手法5種との比較により，提案手法が全5条件で最高または同等のF1を達成し
（平均F1 = 0.70），特に交絡・重複条件での優位性が顕著であった（EX3）．
\item 実キャンペーンデータ（Dunnhumby, 276Kバスケット）において，
集団レベルの効果が期待される大規模販促キャンペーン5件を外部イベントとして使用し，
56\%のクーポン整合率を達成した（ランダム34\%に対し$p = 0.049$; EX4）．
\end{itemize}

\subsection{制約と限界}

本パイプラインには以下の制約がある．
第一に，イベントやパターンが時間的に密集する環境では精度が劣化する．
複数イベントの時間窓が重複する場合にパターン変化を誤ったイベントに帰属する可能性があり，
高密度なベースライン活性下ではイベント非対応パターンの誤帰属率が上昇する．
第二に，円形シフト置換検定は時系列の弱定常性を仮定する．
強いトレンドや季節性を持つデータでは，ブロック置換などの拡張が必要となりうる．
第三に，アイテム頻度がZipf分布に従う条件では，
平方根正規化により高い精度を達成するが（EX1），
高頻度アイテムの自然変動に起因する偽陽性は完全には排除できない．
弱い信号（低ブースト率）は置換検定で有意なスコアを安定して生成しないため，
信号強度と検出力のトレードオフが存在する．
今後の課題として以下が挙げられる:
(1) 帰属スコアの理論的精緻化（尤度比や回帰係数ベースのスコアリング），
(2) イベント間の時間的重複が密な場合の帰属精度改善，
(3) 非定常時系列に対するブロック置換や季節性除去の導入，
(4) 大規模実データ（小売・Web ログ・医療記録）での外的妥当性評価．
