% ==============================================================
\section{結論}\label{sec:conclusion}
% ==============================================================

本稿では，頻出パターンのサポート変動を外部イベントに帰属させる
\textbf{イベント帰属問題}を形式的に定義し，
FDR制御付き多重検定問題としての3つの構造的課題
（仮説空間の規模，副次パターンによる相関偽陽性，時間的混同）を同定した．
この問題に対する解法として，変化点抽出，帰属スコアリング，
置換検定によるFDR補正，アイテム重複排除からなる
5ステップの統計的帰属フレームワークを構成した．
帰属スコアはランキング用指標に留め，
最終判定を置換検定に委ねる二層構造により，
ヒューリスティックな閾値選択への依存を軽減した．

実験により以下の知見を得た:
\begin{itemize}
\item 語彙内ブースト条件（ベースラインサポートが存在する現実的設定）において，
F1 = 0.75--0.94 の検出精度を達成し，
帰無条件下では全20 seedで偽発見数0を確認した（EX1）．
\item アブレーション分析により，proximity と magnitude の相補性，
およびグローバルBH補正と重複排除の段階的な偽陽性削減効果を実証した（EX2）．
\item 実キャンペーンデータ（Dunnhumby, 276Kバスケット, 30キャンペーン）において，
全キャンペーンに対して管理可能な数（平均4.1件）の帰属を生成した（EX3）．
\end{itemize}

\subsection{制約と限界}

本パイプラインには以下の制約がある．
第一に，イベントやパターンが時間的に密集する環境では精度が劣化する．
複数イベントの時間窓が重複する場合にパターン変化を誤ったイベントに帰属する可能性があり，
高密度なベースライン活性下ではイベント非対応パターンの誤帰属率が上昇する
（DENSE条件でFAR = 0.25, F1 = 0.75）．
第二に，円形シフト置換検定は時系列の弱定常性を仮定する．
強いトレンドや季節性を持つデータでは，ブロック置換などの拡張が必要となりうる．
第三に，EX4の結果が示す通り，アイテム頻度がZipf分布に従う現実的な条件では
精度が低下する（F1 = 0.22--0.40）．
相対的変化率ベースのスコアリングや
アイテム頻度に基づく適応的フィルタリングといった拡張が必要である．
第四に，本パイプラインは2アイテムセット（$|P| = 2$）を対象として設計されている
（\ref{sec:method}節）．
Step~1--4 は任意のパターン長に適用可能だが，
Step~5（重複排除）のProposition~\ref{prop:dedup}は
2アイテムセットの簡約モデル下で証明されており，
$|P| \geq 3$ への拡張には共有比率ベースの統合基準や
閉パターン選択などの設計拡張が必要である．

今後の課題として以下が挙げられる:
(1) 帰属スコアの理論的精緻化（尤度比や回帰係数ベースのスコアリング），
(2) イベント間の時間的重複が密な場合の帰属精度改善，
(3) 非定常時系列に対するブロック置換や季節性除去の導入，
(4) 大規模実データ（小売・Web ログ・医療記録）での外的妥当性評価．
