\section{結論}\label{sec:conclusion}
% ==============================================================

本稿では，頻出パターンのサポート時系列における変動を
外部イベントに統計的に帰属させる\textbf{イベント帰属パイプライン}を提案した．
本手法は，(1) Phase 1（Apriori-window）で得られた密集区間からの変化点抽出，
(2) 交差前後のレベル差による変化量評価，
(3) 時間的近接度・方向整合性・変化量に基づく帰属スコアリング，
(4) 円形シフト置換検定とグローバル Benjamini--Hochberg FDR 補正，
(5) Union-Find クラスタリングによるアイテム重複排除
の5ステップから成る．
なお，本手法は変化点とイベントの\textbf{時間的共起}を検定するものであり，
因果効果を立証するものではない．
出力はさらなる調査のためのスクリーニングツールとして解釈すべきである．

特に，副次パターン問題---真に影響を受けたパターンとアイテムを共有するパターンが
相関した偽陽性を生む問題---に対して，
アイテム共有グラフの連結成分分析に基づく重複排除を導入した．
この後処理ステップにより，合成データ実験では全条件で F1 = 1.0 を達成し，
実データでは偽陽性を96\%削減した．

実験により以下の知見を得た:
\begin{itemize}
\item 語彙内ブースト信号（Type A）に対して，全信号強度で真のパターン--イベント帰属を完全に回復する（EX1）．
\item アブレーションにより各構成要素の不可欠性を実証する（EX2）．
\item 帰無データにおいてFDRが名目水準以下に制御されることを確認する（EX3）．
\item 実データにおいて高い再現率を維持しつつ偽陽性を大幅に抑制する（EX5）．
\item Zipfアイテム頻度分布下ではRecallを維持しつつPrecisionが低下する傾向を確認する（EX6）．
\end{itemize}

\subsection{制約と限界}
\label{sec:limitations}

本パイプラインには以下の制約がある．

\begin{enumerate}
\item \textbf{高密度データでの精度低下}:
ベースラインサポートが既に高いパターン，またはZipf分布下で高頻度アイテムが
関与するパターンでは，ブーストによる相対的変化量が小さくなり，
自然変動との区別が困難となる（retail.txt で Recall = 0.0，
EX6 で $\gamma \geq 1.0$ において Precision 低下）．
相対変化率に基づく効果量スコアリングの導入が緩和策として考えられる．

\item \textbf{弱定常性の仮定}:
円形シフト置換検定は，サポート時系列の変化点分布が
時間的に一様であるという暗黙の仮定に依存する．
強いトレンドや季節性が存在する場合，
帰無分布の妥当性が損なわれる可能性がある．

\item \textbf{Zipf分布下でのPrecision低下}:
EX6 で示したように，アイテム頻度の偏りが大きい場合に
Precision が低下する．
これは高頻度アイテムペアの自然変動が大きく，
偽陽性が増加することに起因する．

\item \textbf{$L = 2$ の設計前提}:
本パイプラインは2-itemset（$|P| = 2$）を前提として設計されており，
3-itemset 以上への拡張には，共有アイテム数に応じた
スコア補正や重複排除戦略の再設計が必要となる．
\end{enumerate}

今後の課題として以下が挙げられる:
(1) $|P| \geq 3$ への拡張と対応する重複排除戦略の設計，
(2) イベント間の時間的重複が密な場合の帰属精度改善，
(3) 高ベースラインサポート下での効果量ベーススコアリングの精緻化，
(4) 大規模実データ（小売・Web ログ・医療記録）での外的妥当性評価．
