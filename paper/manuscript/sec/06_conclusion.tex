\section{結論}\label{sec:conclusion}
% ==============================================================

本稿では，頻出パターンのサポート時系列における変動を
外部イベントに統計的に帰属させる\textbf{イベント帰属パイプライン}を提案した．
本手法は，(1) Phase 1（Apriori-window）で得られた密集区間からの変化点抽出，
(2) 交差前後のレベル差による変化量評価，
(3) 時間的近接度・方向整合性・変化量に基づく帰属スコアリング，
(4) 円形シフト置換検定とグローバル Benjamini--Hochberg FDR 補正，
(5) Union-Find クラスタリングによるアイテム重複排除
の5ステップから成る．
なお，本パイプラインは変化点とイベントの\textbf{時間的共起}を検定するものであり，
因果効果を立証するものではない．出力はさらなる調査のためのスクリーニングツールとして
解釈すべきである．

特に，副次パターン問題---真に影響を受けたパターンとアイテムを共有するパターンが
相関した偽陽性を生む問題---に対して，
アイテム共有グラフの連結成分分析に基づく重複排除を導入した．
この後処理ステップにより，合成データ実験では全条件で高精度の検出を達成し，
実データでは偽陽性を96\%削減した．

実験により以下の知見を得た:
\begin{itemize}
\item 語彙内ブースト型信号（Type A）・イベント非対応パターン（Type B）・デコイイベント（Type C）を含む
公正な8条件で評価し，全信号強度で F1 $\geq$ 0.95 かつ FAR = 0.0 を達成する（EX1）．
\item ウィンドウサイズ，減衰パラメータ $\sigma$，置換回数に対してパイプラインは頑健である（EX2）．
\item 帰無条件（Type B パターンのみ）下で棄却率が名目水準 $\alpha = 0.10$ 以下に制御される（EX3）．
\item 処理時間はデータサイズ・イベント数・置換回数に対して線形にスケールし，100万トランザクションまで動作を確認した（EX5）．
\item 実データにおいて高い再現率を維持しつつ偽陽性を大幅に抑制する（EX5）．
\item 実キャンペーンデータ（Dunnhumby, 276Kバスケット, 30キャンペーン）において，
全キャンペーンに対して管理可能な数（平均3.0件）の帰属を生成し，
重複排除が88\%の冗長性を除去する（EX6）．
\end{itemize}

\subsection{制約と限界}

本パイプラインには以下の制約がある．

\begin{enumerate}
\item \textbf{高密度劣化}:
複数イベントが時間的に重複する場合，パターンの変化を誤ったイベントに
帰属する，あるいは近接するデコイイベントに偽陽性を生じうる
（EX1d 実験で確認）．
ベースラインサポートが既に高いパターンでは，ブーストによる相対的変化量が
小さくなり検出が困難となる（retail.txt で Recall = 0.0 となった結果が示す通り）．

\item \textbf{弱定常性仮定}:
円形シフト置換検定は，サポート時系列の自己相関構造がシフトに対して
概ね保存されることを暗黙に仮定する．
強いトレンドや季節性がある場合，この仮定が崩れ検定力が低下しうる．

\item \textbf{Zipf 分布下の精度低下}:
アイテム出現確率が Zipf 分布に従う場合，高頻度アイテムの副次パターンが
増加し，重複排除後も一部の偽陽性が残存する可能性がある．

\item \textbf{$L = 2$ 設計仮定}:
本パイプラインは 2-アイテムセット（$|P| = 2$）を前提として設計されている．
$L \geq 3$ への拡張は，Step 5 の重複排除における連結成分構造の複雑化のため，
追加の理論的正当化が必要となる．
\end{enumerate}

今後の課題として以下が挙げられる:
(1) 変化点検出手法の拡張（CUSUM やベイズ変化点検出との比較），
(2) イベント間の時間的重複が密な場合の帰属精度改善，
(3) ベースラインサポートが高いパターンに対する検出力向上のための
効果量ベーススコアリングの精緻化，
(4) 大規模実データ（小売・Web ログ・医療記録）での外的妥当性評価．
