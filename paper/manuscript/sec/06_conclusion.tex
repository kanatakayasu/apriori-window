\section{結論}\label{sec:conclusion}
% ==============================================================

本稿では\textbf{イベント帰属問題}を形式的に定義し，
初の統計的解法として5ステップの帰属パイプラインを構築した．

実験により以下の知見を得た:
\begin{itemize}
\item Zipf分布を含む多様なデータ条件において高い帰属精度を達成し，
帰無条件下では100 seedで平均偽発見率0.070（$\leq \alpha = 0.10$）を達成し，FDR制御を実証した（EX1）．
\item アブレーション分析により，proximity と magnitude の相補性，
およびグローバルBH補正と重複排除の段階的な偽陽性削減効果を実証した（EX2）．
\item 関連手法5種との比較により，提案手法が全5条件で最高または同等のF1を達成し
（平均F1 = 0.68），特に交絡・重複条件での優位性が顕著であった（EX3）．
\item 実データ（Dunnhumby, 276Kバスケット）で全キャンペーンへの帰属生成（カバレッジ100\%）を確認し，適用可能性を示した（EX4）．
\end{itemize}

\subsection{制約と限界}

本パイプラインには以下の制約がある．
第一に，複数イベントの時間窓が重複・高密度な条件では精度が劣化する．
第二に，円形シフト置換検定は弱定常性を仮定し，
強いトレンドや季節性を持つデータではブロック置換などの拡張が必要となりうる．
第三に，弱信号では検出力とのトレードオフが存在する．
今後の課題として，非定常時系列への対応および大規模実データでの外的妥当性評価が挙げられる．
