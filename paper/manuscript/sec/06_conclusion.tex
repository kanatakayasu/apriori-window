% ==============================================================
\section{結論}\label{sec:conclusion}
% ==============================================================

本稿では，頻出パターンのサポート時系列における変動を
外部イベントに統計的に帰属させる\textbf{イベント帰属パイプライン}を提案した．
本手法は，(1) Phase 1（Apriori-window）で得られた密集区間からの変化点抽出，
(2) 交差前後のレベル差による変化量評価，
(3) 時間的近接度・方向整合性・変化量に基づく帰属スコアリング，
(4) 円形シフト置換検定とグローバル Benjamini--Hochberg FDR 補正，
(5) Union-Find クラスタリングによるアイテム重複排除
の5ステップから成る．

特に，副次パターン問題---真に影響を受けたパターンとアイテムを共有するパターンが
相関した偽陽性を生む問題---に対して，
アイテム共有グラフの連結成分分析に基づく重複排除を導入した．
この後処理ステップにより，合成データ実験では全条件で F1 = 1.0 を達成し，
実データでは偽陽性を96\%削減した．

実験により以下の知見を得た:
\begin{itemize}
\item 全信号強度（ブースト係数 0.1--0.8）で真のパターン--イベント帰属を完全に回復する（E1）．
\item ウィンドウサイズ，減衰パラメータ $\sigma$，置換回数に対してパイプラインは頑健である（E2）．
\item 処理時間はデータサイズ・イベント数・置換回数に対して線形にスケールし，100万トランザクションまで動作を確認した（E3）．
\item 実データにおいて高い再現率を維持しつつ偽陽性を大幅に抑制する（E4）．
\item 実キャンペーンデータ（Dunnhumby, 276Kバスケット, 30キャンペーン）において，
全キャンペーンに対して管理可能な数（平均3.0件）の帰属を生成し，
重複排除が88\%の冗長性を除去する（E5）．
\end{itemize}

\subsection{制約と限界}

本パイプラインにはいくつかの制約がある．
第一に，本手法は変化点とイベントの\textbf{時間的共起}を検定するものであり，
因果効果を立証するものではない．「イベントがなければどうなっていたか」という
反事実的問いには答えられず，出力はさらなる調査のためのスクリーニングツールとして
解釈すべきである．
第二に，複数イベントが時間的に重複する場合，パターンの変化を誤ったイベントに
帰属する，あるいは近接するデコイイベントに偽陽性を生じうる
（E1c/E1d 実験で確認）．
第三に，ベースラインサポートが既に高いパターンでは，ブーストによる相対的変化量が
小さくなり検出が困難となる（retail.txt で Recall = 0.0 となった結果が示す通り）．
第四に，パイプラインは $W$, $\theta$, $\sigma$, $\delta$, $\alpha$ の
設定を要するが，実験ではほとんどのパラメータに対して頑健性を確認した．
ただしウィンドウサイズ $W$ の影響は最も大きく，$W = 200$ では
検出が完全に失敗することが判明した（E2）．

今後の課題として以下が挙げられる:
(1) \textbf{効果量スコアリングの導入}:
  現在の絶対変動量に代えて相対変化率（$\Delta s / s_{\text{base}}$）を用い，
  高ベースラインサポートパターンでの検出力を改善する．
(2) \textbf{閉パターン制約の統合}:
  閉パターンのみを検定対象とすることで
  仮説数 $M$ を構造的に削減し，BH補正の検出力を向上させる．
  これにより $\alpha = 0.05$ での安定動作が期待される．
(3) \textbf{CausalImpact との統合}:
  本パイプラインをスクリーニング段階として用い，
  有意な帰属に対してのみ CausalImpact~\cite{brodersen2015causalimpact}等の
  ベースラインモデル推定を適用する二段階アプローチ．
(4) \textbf{イベント重複時の帰属分離}:
  現在の独立スコアリングに代えて，
  重複イベント間の効果を分離する回帰ベースのアプローチの導入．
(5) \textbf{大規模実データでの外的妥当性評価}:
  正解ラベル付きの小売・Web ログ・医療記録データでの検証．
