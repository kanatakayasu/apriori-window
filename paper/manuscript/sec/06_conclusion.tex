\section{結論}\label{sec:conclusion}
% ==============================================================

本稿では，頻出パターンのサポート時系列における変動を
外部イベントに統計的に帰属させる\textbf{イベント帰属パイプライン}を提案した．
本手法は，(1) Phase 1（Apriori-window）で得られた密集区間からの変化点抽出，
(2) 交差前後のレベル差による変化量評価，
(3) 時間的近接度・方向整合性・変化量に基づく帰属スコアリング，
(4) 円形シフト置換検定とグローバル Benjamini--Hochberg FDR 補正，
(5) Union-Find クラスタリングによるアイテム重複排除
の5ステップから成る．

特に，副次パターン問題---真に影響を受けたパターンとアイテムを共有するパターンが
相関した偽陽性を生む問題---に対して，
アイテム共有グラフの連結成分分析に基づく重複排除を導入した．
この後処理ステップにより，合成データ実験では全条件で F1 = 1.0 を達成し，
実データでは偽陽性を96\%削減した．

実験により以下の知見を得た:
\begin{itemize}
\item 語彙内アイテムを用いた現実的な信号注入設計のもと，全信号強度（ブースト係数 0.1--0.8）で真のパターン--イベント帰属を高精度に回復し，Type B パターンへの偽帰属率（FAR）も 0.00 を達成する（EX1）．
\item ウィンドウサイズ，減衰パラメータ $\sigma$，置換回数に対してパイプラインは頑健である（EX2）．
\item 帰無データに対して名目 FDR が正しく制御されることを実験的に確認した（EX3）．
\item 処理時間はデータサイズ・イベント数・置換回数に対して線形にスケールし，100万トランザクションまで動作を確認した（EX4）．
\item 実データにおいて高い再現率を維持しつつ偽陽性を大幅に抑制する（EX5）．
\item 実キャンペーンデータ（Dunnhumby, 276Kバスケット, 30キャンペーン）において，
全キャンペーンに対して管理可能な数（平均3.0件）の帰属を生成し，
重複排除が88\%の冗長性を除去する（EX6）．
\end{itemize}

\subsection{制約と限界}

本パイプラインには以下の制約がある．

\begin{enumerate}
\item \textbf{高密度劣化}: ベースラインサポートが既に高いパターンでは，
ブーストによる相対的変化量が小さくなり検出が困難となる
（retail.txt で Recall = 0.0 となった結果が示す通り）．
効果量ベースのスコアリング（例: $\Delta s / s_{\text{base}}$）が緩和策となりうる．

\item \textbf{弱定常性仮定}: 円形シフト置換検定は，
サポート時系列の時間構造（自己相関）が帰無仮説のもとで周期的に近似可能であることを
暗黙に仮定する．強いトレンドや非定常性が存在する場合，
$p$値の校正精度が低下する可能性がある．

\item \textbf{Zipf分布下での精度低下}: アイテム頻度が Zipf 分布に従う現実的な条件では，
高頻度アイテムのベースライン変動が大きくなり，
Precision が低下する傾向がある．
この点は今後の Zipf ロバスト性実験で定量的に評価する必要がある．

\item \textbf{$|P|=2$ 設計前提}: 本パイプラインは 2-アイテムセットを対象に設計されており，
命題~\ref{prop:dedup}の独立出現モデルは $|P|=2$ に依存する．
3 以上のアイテムからなるパターンへの拡張は今後の課題である．
\end{enumerate}

今後の課題として，上記制約の解消に加え，
(1) イベント間の時間的重複が密な場合の帰属精度改善，
(2) 大規模実データ（小売・Web ログ・医療記録）での外的妥当性評価
が挙げられる．
