\section{結論}\label{sec:conclusion}
% ==============================================================

本稿では，頻出パターンのサポート変動を外部イベントに帰属させる
\textbf{イベント帰属問題}を形式的に定義し，
FDR制御付き多重検定問題としての3つの構造的課題
（仮説空間の規模，副次パターンによる相関偽陽性，時間的混同）を同定した．
この問題に対する解法として，変化点抽出，帰属スコアリング，
置換検定によるFDR補正，アイテム重複排除からなる
5ステップの統計的帰属フレームワークを構成した．
帰属スコアはランキング用指標に留め，
最終判定を置換検定に委ねる二層構造により，
ヒューリスティックな閾値選択への依存を軽減した．

実験により以下の知見を得た:
\begin{itemize}
\item 語彙内ブースト条件（ベースラインサポートが存在する現実的設定）において，
F1 = 0.67--0.92 の検出精度を達成し，8条件で多面的に弁別能力を検証した（EX1）．
\item スコア構成要素のアブレーションにより，proximity と magnitude が
ともに帰属精度に不可欠であり，相補的に機能することを確認した（EX2）．
\item 帰無条件下（植え込み信号なし）で全20 seedにおいて有意帰属数が0であり，
過剰な偽発見を生じないことを経験的に確認した（EX3）．
\item 実キャンペーンデータ（Dunnhumby, 276Kバスケット, 30キャンペーン）において，
全キャンペーンに対して管理可能な数（平均4.1件）の帰属を生成した（EX5）．
\end{itemize}

\subsection{制約と限界}

本パイプラインにはいくつかの制約がある．
第一に，\ref{sec:intro}節で述べた通り，本手法は\textbf{統計的帰属}
（変化点とイベントの時間的関連の検定）であり，
因果効果を立証するものではない．
出力はさらなる因果分析のためのスクリーニングツールとして解釈すべきである．
第二に，複数イベントが時間的に重複する場合，パターンの変化を誤ったイベントに
帰属する可能性がある．
第三に，高密度なベースライン活性下ではType Bパターンの誤帰属率が大幅に上昇する
（DENSE条件でFAR = 0.55, F1 = 0.67）．
多数のパターンとイベントが同時に存在する環境では，
偶発的な時間的近接がノイズ帰属を生み出しやすく，
これは本手法の主要な制約の一つである．
第四に，円形シフト置換検定は時系列の弱定常性を仮定する．
強いトレンドや季節性を持つデータでは，ブロック置換などの拡張が必要となりうる．
第五に，パイプラインは $W$, $\theta$, $\sigma$, $\delta$, $\alpha$ の
設定を要するが，パラメータ感度分析では
$W = 10$--$50$ で安定した検出を維持することを確認した（付録）．
第六に，EX6の結果が示す通り，アイテム頻度がZipf分布に従う現実的な条件では
精度が低下する（F1 = 0.22--0.40）．
本手法は現状では，アイテム頻度が中程度に均質なデータ，
または大規模イベントによる明確なサポート変動がある条件を主対象とする．
Zipf分布下での適用には，相対的変化率ベースのスコアリングや
アイテム頻度に基づく適応的フィルタリングといった拡張が必要である．

今後の課題として以下が挙げられる:
(1) 帰属スコアの理論的精緻化（尤度比や回帰係数ベースのスコアリング），
(2) イベント間の時間的重複が密な場合の帰属精度改善，
(3) 非定常時系列に対するブロック置換や季節性除去の導入，
(4) 大規模実データ（小売・Web ログ・医療記録）での外的妥当性評価．
