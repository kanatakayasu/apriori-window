\section{結論}\label{sec:conclusion}
% ==============================================================

本稿では，頻出パターンのサポート時系列における変動を
外部イベントに統計的に帰属させる\textbf{イベント帰属パイプライン}を提案した．
本手法は，(1) Phase 1（Apriori-window）で得られた密集区間からの変化点抽出，
(2) 交差前後のレベル差による変化量評価，
(3) 時間的近接度・方向整合性・変化量に基づく帰属スコアリング，
(4) 円形シフト置換検定とグローバル Benjamini--Hochberg FDR 補正，
(5) Union-Find クラスタリングによるアイテム重複排除
の5ステップから成る．

特に，副次パターン問題---真に影響を受けたパターンとアイテムを共有するパターンが
相関した偽陽性を生む問題---に対して，
アイテム共有グラフの連結成分分析に基づく重複排除を導入した．

実験により以下の知見を得た:
\begin{itemize}
\item 語彙内ブースト条件（ベースラインサポートが存在する現実的設定）において，
F1 = 0.54--0.85 の検出精度を達成し，8条件で多面的に弁別能力を検証した（EX1）．
\item スコア構成要素のアブレーションにより，magnitude が支配的な帰属要素であり，
proximity が副次的に寄与することを確認した（EX2）．
\item 実キャンペーンデータ（Dunnhumby, 276Kバスケット, 30キャンペーン）において，
全キャンペーンに対して管理可能な数（平均3.0件）の帰属を生成し，
重複排除が88\%の冗長性を除去する（EX5）．
\end{itemize}

\subsection{制約と限界}

本パイプラインにはいくつかの制約がある．
第一に，本手法は変化点とイベントの\textbf{時間的共起}を検定するものであり，
因果効果を立証するものではない．「イベントがなければどうなっていたか」という
反事実的問いには答えられず，出力はさらなる調査のためのスクリーニングツールとして
解釈すべきである．
第二に，複数イベントが時間的に重複する場合，パターンの変化を誤ったイベントに
帰属する可能性がある（OVERLAP条件でF1 = 0.63）．
第三に，高密度なベースライン活性下ではType Bパターンの誤帰属率が上昇する
（DENSE条件でFAR = 0.55）．
第四に，パイプラインは $W$, $\theta$, $\sigma$, $\delta$, $\alpha$ の
設定を要するが，パラメータ感度分析では
$W = 10$--$50$ で安定した検出を維持することを確認した（付録）．

今後の課題として以下が挙げられる:
(1) 変化点検出手法の拡張（CUSUM やベイズ変化点検出との比較），
(2) イベント間の時間的重複が密な場合の帰属精度改善，
(3) ベースラインサポートが高いパターンに対する検出力向上のための
効果量ベーススコアリングの精緻化，
(4) 大規模実データ（小売・Web ログ・医療記録）での外的妥当性評価．
