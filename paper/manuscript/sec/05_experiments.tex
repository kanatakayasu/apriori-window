% ==============================================================
\section{実験}\label{sec:experiments}
% ==============================================================

\subsection{実験設計}

\subsubsection{合成データ生成}

ベース語彙200アイテム（各アイテムの出現確率 $p_{\text{base}} = 0.03$），
$N = 5{,}000$ トランザクションの合成データを生成する．
外部イベントの効果は\emph{植え込み信号}としてモデル化する．
3種類の信号を1つのデータセットに混在させることで，
パイプラインの弁別能力を多面的に評価する:

\noindent\textbf{Type A}（語彙内ブースト）:
ベース語彙内のアイテムペアを使用し，
イベント期間外にも $p_{\text{base}}$ でベースラインサポートを持たせる．
イベント活性期間中に確率 $\beta$ で全パターンアイテムを同時挿入し
サポートを上昇させる．
magnitude単独では不十分な\textbf{現実的条件}を構成する．

\noindent\textbf{Type B}（イベント非対応密集パターン）:
特定期間にサポートが上昇するが，いずれのイベントとも
対応しないパターン．季節性やトレンドを模擬し，
パイプラインが\textbf{無関係な変動を棄却}できるかを検証する．

\noindent\textbf{Type C}（デコイイベント）:
パターン変動と無関係なイベント．

\subsubsection{評価指標}

\begin{itemize}
\item \textbf{P/R/F1}: 予測と正解の厳密一致に基づく
    Precision，Recall，F1
\item \textbf{偽帰属率 (FAR)}: Type Bパターンがイベントに誤帰属された割合
\end{itemize}

\subsubsection{評価方針}

本研究が定式化するイベント帰属問題（定義~\ref{def:attribution}）は，
数千のパターンサポート時系列を複数の外部イベントに同時に帰属させ，
多重検定を制御しつつ副次パターンの偽陽性を排除するタスクである．
表~\ref{tab:related_comparison}に示した通り，
既存手法はいずれもこの問題を直接解くものではない．
そこで本実験では以下の4つの評価軸を用いる．
第一に，提案手法単体の帰属精度を多様なデータ条件で評価する（EX1）．
第二に，パイプラインの各構成要素を段階的に除去する
\textbf{アブレーション分析}（EX2）により，
各ステップの必要性と寄与を実証する．
第三に，\ref{sec:related}節の関連手法（表~\ref{tab:related_comparison}）
を本パイプラインのSteps~1--3--4に導入する
\textbf{手法比較実験}（EX3）により，
提案手法の各ステップの選択が最適であることを検証する．
第四に，実データでの探索的検証を行う（EX4）．

\subsection{EX1: コア帰属精度}
\label{sec:exp_ex1}

\textbf{RQ}: パイプラインはイベント起因の変動を正しく帰属し，
イベント無関係の密集パターンを棄却できるか？
多様なデータ特性（信号強度，構造条件，アイテム頻度分布）に対する頑健性を評価する．
実世界のトランザクションデータはアイテム頻度がZipf分布に
従うケースが多いため~\cite{fournier2017survey}，
Zipf分布（$\alpha=1.0$）の合成データに平方根正規化を適用して評価する．

\paragraph{信号強度}
Type A（語彙内ブースト3パターン），
Type B（イベント非対応2パターン），
Type C（デコイ2件）を混在させ，
$\beta \in \{0.2, 0.3, 0.5\}$ で5 seed実行した
（図~\ref{fig:ex1a_signal_strength}）．

\begin{figure}[t]
\centering
\includegraphics[width=\linewidth]{fig/ex1a_signal_strength.pdf}
\caption{EX1a: 信号強度 $\beta$ と帰属精度の関係（5 seeds平均，Zipf-1.0，$\sqrt{}$正規化）．
$\beta = 0.3$ でF1が最大となり，$\beta$ の増加に伴いPrecisionが低下する一方，FARが上昇する．}
\label{fig:ex1a_signal_strength}
\end{figure}

全$\beta$条件でF1 $\geq$ 0.69を達成している．
$\beta = 0.2$ではPrecision = 0.78と高いが，Recall = 0.73とやや低下する．
$\beta = 0.3$でRecall = 0.93, F1 = 0.77と最も均衡の取れた精度を示す．
$\beta = 0.5$ではRecall = 0.87だがPrecision = 0.57に低下する．
強いブーストは上位集合パターンの過剰検出を招き，
Zipf分布下ではその影響がより顕著になるためである．
なお$\beta = 0.1$は振幅フィルタ（$\delta = 10$）により検定対象外となる．
$\delta$を下げれば弱い信号の検出力は向上するが，
振幅の小さいノイズパターンが大量に検定対象に入り
BH補正の負荷が増大するトレードオフがある．
$\beta \geq 0.2$でRecall $\geq$ 0.73を達成していることから，
$\delta = 10$は実用的な閾値と言える．

\paragraph{構造条件}
デフォルト設定（$\beta = 0.3$, Zipf-1.0, $\sqrt{}$正規化）で
4つの構造条件を評価した結果を図~\ref{fig:ex1b_structural}に示す．

\begin{figure}[t]
\centering
\includegraphics[width=\linewidth]{fig/ex1b_structural.pdf}
\caption{EX1b: 構造条件別の帰属精度（5 seeds平均，Zipf-1.0，$\sqrt{}$正規化）．
全条件でF1 $\geq$ 0.70を達成．CONFOUND・DENSEではFARが上昇するがRecallは維持される．}
\label{fig:ex1b_structural}
\end{figure}

OVERLAPではF1 = 0.84, FAR = 0.00と高い精度を達成し，
イベント区間が重複する条件でも誤帰属を完全に棄却できている．
CONFOUNDではType Bパターンがイベント近傍に配置される交絡条件で
FAR = 0.40と上昇するが，Recall = 0.93を維持しており，
真の信号の検出力は損なわれていない．
DENSEではFAR = 0.45と上昇するが，Recall = 0.93と高い検出力を維持している．
SHORTでは短期間イベント（持続期間80）でFAR = 0.30であり，
Recall = 0.87は維持されている．
全条件でF1 $\geq$ 0.70を達成しており，
変化点ベースのスコアリングと円形シフト置換検定が
多様な構造的課題に対して有効に機能することが確認された．

\paragraph{アイテム頻度分布への頑健性}
より強い偏りを持つZipf分布（$\alpha = 1.5$）でも
デフォルト設定（$\beta = 0.3$，$\sqrt{}$正規化）において
F1 = 0.79, FAR = 0.10を達成しており，
Zipf-1.0（F1 = 0.77）と同等以上の精度を維持している．
$\alpha = 1.5$ではヘッドアイテムの出現頻度がさらに集中するため
ノイズ水準が上昇するが，平方根正規化がこの効果を吸収し，
帰属精度の低下を防いでいる．

\paragraph{帰無条件検証}
植え込み信号を一切含まない帰無データ（EX1と同一の生成条件，
5件のランダムイベント，持続期間300）で20 seed実行した結果，
全seedにおいて有意帰属数は0であった．
BH手続きと円形シフト置換検定の組み合わせが
帰無仮説下のp値を十分に大きく保ち，
過剰な偽発見を生じないことが経験的に確認された．
名目$\alpha = 0.10$に対して偽発見率0は，
本手法が保守的に制御していることを示す．
この保守性は，置換検定のp値離散性
（$B = 5000$での最小p値 $\approx 2 \times 10^{-4}$）と
BH補正の相乗効果に起因する．
保守的な制御は偽発見リスクの最小化において実用上望ましいが，
弱い信号の検出力を犠牲にする可能性がある．
実際，$\beta = 0.2$条件でRecall = 0.73と若干の低下が観察されており
（図~\ref{fig:ex1a_signal_strength}），これは保守的制御のコストと解釈できる．

\subsection{EX2: アブレーション分析}
\label{sec:exp_ex2}

\textbf{RQ}: スコア構成要素およびパイプラインの各ステップは精度にどう寄与するか？
EX1と同様にZipf分布データ（$\sqrt{}$正規化）を使用する．

\paragraph{スコア構成要素}

各構成要素の必要性を実証する2つのシナリオを設計し，
全3アブレーションモードを各シナリオに適用する（2シナリオ $\times$ 3モード $\times$ 5 seeds = 30条件）:

\noindent\textbf{Scenario A} (prox必要):
パターン $(5, 15)$ に2つの密集区間 --- 1つはイベント近傍（因果的），
1つは遠方（偶発的）．proxなしでは両者に等しいスコアが付与される．

\noindent\textbf{Scenario B} (mag必要):
パターン $(5, 15)$ が2イベントの影響を受けるが，
ブースト量が $\beta=0.5$ と $\beta=0.1$ で大小異なる．
magなしでは両者が同等に扱われる．

結果を表~\ref{tab:ex2a}に示す．

\begin{table}[t]
\caption{EX2a 結果: シナリオ別アブレーション（5 seeds平均F1，Zipf-1.0，$\sqrt{}$正規化）}
\label{tab:ex2a}
\centering
\begin{tabular}{lcc}
\toprule
構成 & Scenario A & Scenario B \\
\midrule
Full: $\text{prox} \times \text{mag}$ & \textbf{1.00} & \textbf{0.80} \\
mag のみ（prox除去） & 0.00 & 0.00 \\
prox のみ（mag除去） & 0.80 & 0.47 \\
\bottomrule
\end{tabular}
\end{table}

表~\ref{tab:ex2a}から，\textbf{proximity が帰属に不可欠}であることが明らかである．
mag のみ（prox除去）の構成は時間的対応付け情報を一切持たず，
全変化点--イベントペアに等しいスコアが付与される結果，
\textbf{両シナリオでF1 = 0.00}に崩壊する．
prox のみ（mag除去）ではScenario Aで F1 = 0.80，Scenario Bで F1 = 0.47 と部分的に機能するが，
変化量情報の欠如により精度は不十分である．
Full構成はScenario AでF1 = 1.00，Scenario BでF1 = 0.80を達成し，
両構成要素の相補性を実証している．
乗算構成がいずれかの構成要素が0に近ければスコア全体を抑制する
排他的フィルタとして機能し，偽陽性の抑制に寄与する．

\paragraph{パイプラインステップの寄与}
ナイーブベースライン（Steps 1--3のみ: 変化点検出 + スコアリング，
閾値以上の候補をすべて出力）と，パイプラインの各ステップを
段階的に追加した3つの変種を比較する:

\begin{itemize}
\item \textbf{Naive}: Steps 1--3のみ（検定なし）
\item \textbf{+PermTest}: per-pattern 置換検定を追加（グローバル補正なし）
\item \textbf{+BH}: グローバルBH補正を追加
\item \textbf{Full}: Union-Find重複排除を追加（= 完全パイプライン）
\end{itemize}

EX1のデータ（$\beta = 0.3$, CONFOUND, DENSE; Zipf-1.0, $\sqrt{}$正規化）を使用し，
各条件 $\times$ 5 seedsで評価した（図~\ref{fig:ex2b_ablation}）．

\begin{figure}[t]
\centering
\includegraphics[width=\linewidth]{fig/ex2b_ablation.pdf}
\caption{EX2b: パイプラインステップの段階的追加による精度変化（5 seeds平均，Zipf-1.0，$\sqrt{}$正規化）．
(a) F1: BH補正と重複排除の追加で大幅に改善．
(b) FAR: 各ステップが段階的に偽帰属率を削減する．}
\label{fig:ex2b_ablation}
\end{figure}

Naiveは全条件でRecall = 1.00を達成するが，
Precision が極めて低く偽陽性を大量に出力する（F1 $\leq$ 0.03）．
スコア閾値のみでは偽陽性の制御が不可能であることを示す．

\textbf{+PermTest}（per-pattern置換検定）は出力数をわずかに削減するが，
パターン単位の補正ではグローバルな多重検定の負担に対応できず，
FPの削減効果は限定的である．

\textbf{+BH}（グローバルBH補正）が最大の偽陽性削減効果を持つ．
BH手続きがグローバルなp値順位に基づきFDRを制御することで，
個別パターンの検定では見逃される弱い偽陽性を効果的に棄却する．
$\beta=0.3$条件ではNaiveの270件がBH補正後に5件まで削減される．

\textbf{Full}（+重複排除）は残存する冗長パターン
（同一イベントに帰属されたアイテム重複・部分集合パターン）を除去する．
$\beta=0.3$条件ではF1が0.71から0.77に改善し，
CONFOUND条件でもFARを1.00から0.40に削減する．
全条件においてFullが最高のF1を達成しており，
各ステップが段階的かつ相補的に偽陽性を削減していることが確認された．

\subsection{EX3: 関連手法との比較}
\label{sec:exp_ex3}

\textbf{RQ}: パイプラインのSteps~1--3--4を関連手法で置き換えた場合，
提案手法の構成要素は最適か？

表~\ref{tab:related_comparison}に示した関連手法群のうち，
イベントとアウトカムの関連を検定する手法を選定し，
帰属判定の核となるSteps~1（変化点抽出），3（帰属スコアリング），
4（統計検定）を以下の5手法で置き換える．
全手法はPhase~1（Apriori-window），振幅フィルタ，BH補正，
Union-Find重複排除（Step~5）を共有し，
帰属判定部分のみが異なる．
ECAのみ例外として提案手法のStep~1（変化点抽出）を使用し，
Steps~3--4を置き換える．

\begin{enumerate}
\item \textbf{Wilcoxon}~\cite{mann1947test}:
  イベント期間中と期間外のサポート分布をrank-sum検定で比較（\ref{sec:related}節）．
\item \textbf{CausalImpact}~\cite{brodersen2015causalimpact}:
  イベント前データから局所水準モデルで反事実を推定し，累積逸脱を検定（\ref{sec:related}節）．
\item \textbf{ITS}~\cite{bernal2017interrupted}:
  セグメント回帰で介入前後の水準変化を推定し，水準変化係数を検定（\ref{sec:related}節）．
\item \textbf{EventStudy}~\cite{mackinlay1997event}:
  イベント前の平均・標準偏差から異常サポートを算出し，累積異常サポートを$z$検定（\ref{sec:related}節）．
\item \textbf{ECA}~\cite{donges2016event}:
  提案手法のStep~1で抽出した変化点とイベントの時間的一致を二項検定で評価（\ref{sec:related}節）．
\end{enumerate}

EX1と同一のデータ（$\beta = 0.3$, Zipf-1.0, $\sqrt{}$正規化）および
4つの構造条件を使用し，各条件5 seedsで評価した結果を
図~\ref{fig:ex3_method}に示す．

\begin{figure}[t]
\centering
\includegraphics[width=\linewidth]{fig/ex3_method.pdf}
\caption{EX3 結果: 関連手法との比較（5 seeds平均）．
全手法はPhase~1・振幅フィルタ・BH補正・重複排除を共有する．
(a) F1: 提案手法は4条件で最高．$\beta$=0.3ではITSが僅差で上回るがFARは提案手法が優位．
(b) FAR: 提案手法は全条件で最低水準の偽帰属率を維持（ECAの低FARは出力数の少なさに起因）．}
\label{fig:ex3_method}
\end{figure}

提案手法は5条件中4条件でF1が最高であり，
特にCONFOUND条件ではF1 = 0.81と他の全手法を大幅に上回る
（次点ITS: 0.39）．
変化点のタイミング情報を活用するproximityスコアが，
イベント近傍に偶然配置された非対応パターンを効果的に弁別する．
$\beta$=0.3条件ではITSがF1 = 0.81と提案手法（0.77）に匹敵するが，
FAR = 0.40と偽帰属率が高い（提案手法: 0.10）．
CONFOUND条件ではITSのF1は0.39に崩壊し，FAR = 0.90に達する．
セグメント回帰はイベント前後のトレンド変化を検定するため，
交絡的なサポート変動に脆弱である．
CausalImpactとWilcoxonは全条件でF1が低く，
特にCONFOUND条件ではF1 $\leq$ 0.22に留まる．
両手法ともFARが0.50--1.00と高く，
イベント非対応パターンの棄却能力に欠ける．
ECAは変化点とイベントの時間的一致を二項検定で評価するが，
Recall $\leq$ 0.20と極めて低い．
密集パターンが複数の変化点を持つ場合に一致確率が希釈されるためである．
なおECAのFARが低いのは棄却能力の高さではなく，
出力数自体が極めて少ない（条件あたり0--5件）ことに起因する．
真の信号をほぼ検出できないため，偽帰属の機会自体が生じない．

総合的に，提案手法はF1とFARの両面で最良のバランスを示す．
FAR平均は提案手法が0.25であるのに対し，
次点のITSは0.54，EventStudyは0.59と大きく劣る．
特に交絡条件（CONFOUND, DENSE）で他手法との差が顕著であり，
変化点の時間的近接度と変化量を組み合わせた帰属スコアの有効性が確認された．

\subsection{EX4: 実データ検証}
\label{sec:exp_ex4}

\textbf{RQ}: 実世界データで意味のある結果を生成できるか？

Dunnhumby ``The Complete Journey'' データセット~\cite{dunnhumby2014complete}の
276{,}484件の購買バスケットを用いた探索的分析．
実データのアイテム頻度はZipf的分布に従うため，平方根正規化を使用する．

\paragraph{外部イベントの選定}
本パイプラインは集団レベルのサポート変動を分析対象とする（定義~\ref{def:attribution}）．
Dunnhumby データセットには30件の販促キャンペーンが記録されているが，
このうちターゲット型施策（TypeB/C: クーポン対象商品数2{,}000件未満）は
特定世帯向けの個別クーポン配布であり，
集団レベルのサポート時系列に検出可能な影響を与えることが期待できない．
そこで，クーポン対象商品数3{,}000件以上の
\textbf{大規模販促キャンペーン5件}（C8, C13, C18, C26, C30;
対象商品数3.7K--38.2K件）を外部イベントとして使用する．

\paragraph{クーポン整合性による部分的検証}
Dunnhumby データセットには各キャンペーンのクーポン対象商品リスト（coupon.csv）が
付属しており，帰属結果の部分的な検証が可能である．
帰属されたパターン $(a, b)$ の少なくとも1つのアイテムが
当該キャンペーンのクーポン対象商品に一致する場合を
\textbf{クーポン整合的}（coupon-consistent）と定義する．
帰属された18件のうち10件がクーポン整合的であり，
\textbf{整合率56\%}を達成した．

この整合率の意味を評価するため，ランダム帰属のベースラインを解析的に計算する．
パターン $(a, b)$ がランダムに帰属された場合，
少なくとも1アイテムがクーポン対象商品に含まれる確率は
$P(\text{match}) = 1 - \binom{N - |C_c|}{2} / \binom{N}{2}$
（$N$: 全商品数92{,}339，$|C_c|$: キャンペーンのクーポン対象数）であり，
大規模キャンペーン5件の平均では$P(\text{random}) = 0.34$である．
提案手法の整合率0.56は\textbf{ランダムの1.6倍}に相当する．
二項検定により，18件中10件の整合は
帰無仮説（$p = 0.34$）のもとで$p = 0.049$（片側）であり，
帰属が偶然を超える整合性を持つことが統計的に確認された．

\paragraph{クーポンリストの限界}
ただし，クーポン整合率は帰属精度の正確な指標ではない．
クーポン対象商品リストはキャンペーンの\textbf{直接的対象}のみを記録しており，
キャンペーンがもたらす間接的な購買行動変化を捕捉しない．
例えば，ビールの販促により購入頻度が上昇した顧客が
スナックやチーズを併買する場合（cross-selling），
パターン（スナック, チーズ）のサポート変動は
正しい帰属であるがクーポンリストには記録されない．
同様に，店舗全体の陳列変更やPOP掲示など
クーポン以外の販促効果も対象外である．
すなわち，クーポン非整合の帰属が誤帰属を意味するとは限らず，
整合率56\%は帰属精度の\textbf{下界}と解釈すべきである．
正確な評価にはキャンペーン効果の全容
（直接効果・間接効果・店舗施策）を記録した
正解データが必要であり，これは本データセットの制約である．
