\subsection{データ形式と乱数設定}
\label{sec:supp_data_format}

生成データ（\texttt{*\_seedX.txt}）は 1 行 1 トランザクション形式で，
同一行内のバスケットを区切り文字 \texttt{" | "} で連結する．
例: \texttt{1 4 9 | 2 8}

乱数は Python の \texttt{random.Random(seed)} を用い，
seed は A1/A2 で \texttt{0..4}，A4 で \texttt{0..2} を使用する．
したがって，同一 seed・同一パラメータ・同一 Python 実行系で
トランザクションファイルは決定的に再生成される．

\subsection{Ground Truth の再計算}
\label{sec:supp_gt}

GT 判定は \texttt{experiments/gen\_synthetic.py} の
\texttt{compute\_ground\_truth()} を用いて実施する．
多項目集合 $S$（$2 \leq |S| \leq max\_length$）に対して，
\begin{itemize}
    \item \texttt{true\_support(S)}: 同一バスケット内で $S$ が成立した回数
    \item \texttt{txn\_support(S)}: トランザクション単位（バスケット横断可）で $S$ が成立した回数
\end{itemize}
を計数し，
\texttt{true\_support(S) < min\_support <= txn\_support(S)}
を満たす場合を spurious と判定する．
SPR は本文 Eq.~(\ref{eq:spr}) に対応し，
分母を \texttt{txn\_support >= min\_support} の多項目集合数とした比率として計算する．
