# Main Manuscript

The primary manuscript source is located at `paper/manuscript/` (repository root).

## Compilation

```sh
cd paper_basket && latexmk -pdf -interaction=nonstopmode main.tex
```

## Structure

- `paper/manuscript/main.tex` - Main document
- `paper/manuscript/sec/` - Section files (01_intro.tex through 07_supplement.tex)
- `paper/manuscript/refs.bib` - Bibliography
- `paper/manuscript/fig/` - Figures
- `paper/manuscript/alg/` - Algorithm pseudocode
