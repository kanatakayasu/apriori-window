Multi-Dimensional Dense Region Mining

From Intervals to Level Sets of Support Surfaces

KDD 2027 / ICDM 2026 Paper C

Key Insight

Pattern support is not just a number or a time series -- it is a surface over multiple contextual dimensions (time, space, demographics). Dense regions are the connected components of its superlevel set, generalizing 1D dense intervals to arbitrary dimensions and shapes.
📊

Support Surface

A function SP(t, v) counting pattern support within a sliding window at each point in the multi-dimensional domain.

🌍

Dense Region

Connected component of the superlevel set {(t, v) : SP(t, v) ≥ θ}. Generalizes 1D intervals to arbitrary shapes.

⚡

Sweep Surface

Efficient algorithm combining prefix-sum computation with union-find, achieving O(G · α(G)) complexity.

Theoretical Contributions

Theorem 1: Dense Region Containment

For any pattern P and subset Q = P \ {i}: every dense region of P is contained within some dense region of Q. This extends Apriori anti-monotonicity to multi-dimensional space, enabling candidate region pruning.

Theorem 2: Dimension Decomposability

When the support surface factorizes as SP(t, v) = f0(t) · ∏ fj(vj), dense region detection decomposes into k+1 independent 1D problems, each solvable in O(Tj) time.

Algorithm

function SweepSurfaceDetect(S, θ, shape):
  U ← UnionFind(|grid|)
  above ← {}
  for t = 0 to T0 - 1:
    for each v in slice(t):
      if S(t, v) ≥ θ:
        above ← above ∪ {(t, v)}
        for each neighbor n:
          if n ∈ above: U.Union((t,v), n)
  return ConnectedComponents(U, above)

// Complexity: O(G · α(G)) where G = grid size

How We Compare

Existing Approaches

  • SaTScan: circular/elliptical windows only
  • DenseAlert/M-Zoom: axis-aligned blocks only
  • DBSCAN: point data, not support functions
  • Co-location mining: no temporal dynamics
  • 1D dense intervals: temporal only

Our Approach

  • Arbitrary region shapes (superlevel sets)
  • Works on support surfaces (function space)
  • Multi-dimensional Apriori pruning
  • Dimension decomposability theory
  • Generalizes 1D to any number of dimensions
