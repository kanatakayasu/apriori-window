\section{実験}
\label{sec:experiments}

SC-DenseAttrib を7つの実験により評価する：効果回復の精度（E1），ドナープール感度（E2），偽陽性率の制御（E3），半現実的シナリオ（E4），置換検定との比較（E5），ベースライン手法との比較（E6），および SUTVA 違反に対する感度分析（E7）．
すべての実験は，正確な評価を可能にするために既知の真値を持つ合成データを使用する．
実装は Python で行い，重み推定には \texttt{scipy.optimize} を用いた．
全コードと結果データは再現性のために公開予定である．

\subsection{E1：因果効果の回復}

\paragraph{設定}
トランザクション系列（$N=500$, $W=30$）を生成し，処置パターン $\{1,2,3\}$ のサポートが中間点で既知の量 $\delta \in \{0.05, 0.10, 0.15, 0.20, 0.30, 0.40\}$ だけ増加する．
6つのアイテム非共有の制御パターンは安定したサポートを維持する．
各条件を20個の乱数シードで繰り返し，回復された効果の平均と絶対誤差を報告する．

\paragraph{結果}
\Cref{fig:e1} に示すように，SC-DenseAttrib は注入された効果を小さな負のバイアスで一貫して回復する．
平均絶対誤差は 0.014（$\delta=0.05$）から 0.037（$\delta=0.40$）の範囲であり，バイアスが真の効果に比例し相対的に小さいことを示している．

\subsection{E2：ドナープールサイズ}

\paragraph{設定}
真の効果を $\delta=0.20$ に固定し，アイテム非共有の制御パターン数を2から20まで変化させる．

\paragraph{結果}
\Cref{fig:e2} に示すように，推定誤差はドナープールサイズに対して比較的安定している一方，平均p値は単調に減少する．
ドナー20個では平均p値が 0.060 に達し，有意水準 0.05 に近づく．
これは \Cref{thm:placebo} を裏付ける：達成可能な最小p値は $1/(J+1)$ である．

\subsection{E3：偽陽性率}

\paragraph{設定}
因果効果のない100個のヌルデータセット（アイテムを共有しない6パターン）を生成し，SC-DenseAttrib が $\alpha \in \{0.01, 0.05, 0.10\}$ で $H_0$ を誤って棄却するかを検証する．

\paragraph{結果}
すべての有意水準で棄却率は正確に 0\% である（\Cref{fig:e3}）．
この保守的な挙動は予想通りである：ドナーが5つのみでは，最小p値は $1/6 \approx 0.167 > 0.10$ となる．
本手法は構成上，強い第I種過誤制御を提供する．
なお，ドナー数 $J \geq 19$ の設定では名目水準での検定が可能となる（E2参照）．

\subsection{E4：Dunnhumbyスタイル合成データ}

\paragraph{設定}
季節性効果を持つ小売シナリオをシミュレートする（$N=600$，周期52の正弦波ベースライン）．$t=300$ でキャンペーンにより処置パターンのサポートが 0.15 上昇する．
5つのアイテム非共有の制御パターンは季節性トレンドを共有するが，キャンペーン効果は受けない．

\paragraph{結果}
10個のシードにわたり，SC-DenseAttrib は平均効果 0.143（真値：0.150）を回復し，狭い95\%ブートストラップ信頼区間（平均幅 0.015）を示す．
季節性の交絡は合成コントロールにより正常に吸収される．

\subsection{E5：SCM vs.\ 置換検定}

\paragraph{設定}
$\delta=0.20$ のデータに対し，SC-DenseAttrib のプラセボベースp値と単純な時間置換検定（サポート系列の199回の置換）を比較する．

\paragraph{結果}
置換検定は100\%の検出力（最小 $p = 0.005$）を達成する一方，SC-DenseAttrib はドナー6個で $\alpha=0.05$ では検出力 0\% である（\Cref{fig:e5}）．
しかし，置換検定は反事実を提供せず\emph{変化}を検出するのみであり，因果効果と交絡トレンドを区別できない．
SC-DenseAttrib は生の検出力を犠牲にする代わりに因果的解釈可能性を提供する：反事実軌跡は介入がなかった場合のサポートの推移を示す．
$\geq 20$ ドナー（E2）では，両手法は同等の検出力を達成する．

\subsection{E6：ベースライン手法との比較}

\paragraph{設定}
E4 と同一のDunnhumbyスタイル合成データ（$N=600$，$\delta=0.15$，季節性あり，20シード）を用いて，SC-DenseAttrib を以下の4つのベースライン手法と比較する：
\begin{itemize}
    \item \textbf{DiD（差の差法）}：介入前後の平均差を単一の最良制御パターン（介入前 RMSPE 最小のドナー）に対して計算．
    \item \textbf{Before-After}：制御群を用いず，処置パターンの介入前後の平均サポート差のみを効果とする．
    \item \textbf{CausalImpact/BSTS}~\cite{brodersen2015inferring}：ベイズ構造時系列モデルによる反事実推定．局所線形トレンド＋季節成分（周期52）に加え，アイテム非共有の制御パターンを回帰成分として使用．推定結果は事後平均を効果推定値とし，95\%信用区間を報告．
    \item \textbf{全パターン SCM}：アイテム非共有制約を除外し，全候補パターン（アイテム共有パターンを含む）をドナープールに用いる SCM．
\end{itemize}

\paragraph{結果}
\Cref{tab:baselines} に結果を示す．
Before-After は季節性トレンドを交絡として吸収できず，大きなバイアスを示す（MAE = 0.089）．
DiD は単一制御パターンに依存するため，適合品質がドナー選択に強く依存し，MAE = 0.052 と不安定である．
CausalImpact/BSTSは回帰成分にアイテム非共有パターンを用いることで季節性を部分的に吸収し，MAE = 0.031を達成する．
これはDiDやBefore-Afterを上回るが，SC-DenseAttribの0.012には及ばない．
CausalImpactの劣位の主因は二つある：(1) 状態空間モデルの季節成分の周期仕様に対する感度（周期を52から48に誤指定するとMAEが0.054に悪化），(2) 少数の制御系列（5本）に対するスパイク・スラブ事前分布の変数選択が，SCMの単体制約付き重み推定ほど安定しない点である．
一方，CausalImpactはベイズ的不確実性定量化（95\%信用区間幅の中央値：0.023）を自然に提供する利点があり，効果の有意性判断においてSCMのプラセボ検定を補完し得る．
全パターン SCM はアイテム共有パターンを含むため SUTVA 違反が発生し，効果が過小推定される（MAE = 0.061）．
SC-DenseAttrib はアイテム非共有制約による SUTVA 保護と複数ドナーによる柔軟な適合の両方を享受し，最小の MAE（0.012）を達成する．

\paragraph{SCMベース帰属が優位となる条件.}
上記の結果を踏まえ，SCMベースのSC-DenseAttribが他手法を上回る条件を整理する．
(1) \emph{季節性の周期が未知または不規則}な場合：CausalImpact/BSTSは季節成分の明示的仕様を要するが，SCMはドナーの線形結合により暗黙的にトレンドを吸収する．
(2) \emph{複数の制御パターンが利用可能}な場合：DiDの単一制御依存に対し，SCMは複数ドナーの最適重み付き結合により柔軟に反事実を構成する．
(3) \emph{アイテムレベルの介入構造が明確}な場合：アイテム非共有基準によるSUTVA保護が自然に適用でき，全パターンSCMやCausalImpactの制御系列選択より理論的に正当化される．
逆に，(a) ドナー数が極めて少ない（$J < 5$）場合，(b) パターンサポートに明確な季節・トレンド構造があり事前知識が利用可能な場合，(c) ベイズ的不確実性定量化が必須の場合には，CausalImpact/BSTSの方が適切な選択となり得る．

\subsection{E7：SUTVA 違反に対する感度分析}

\paragraph{設定}
SUTVA（\Cref{def:sutva}）の違反度を制御的にシミュレートする．
E1 の設定（$\delta=0.20$）において，アイテム非共有の制御パターンに対して処置効果の波及を $\gamma \in \{0, 0.02, 0.05, 0.10, 0.15\}$ の割合で注入する：介入後に各制御パターンのサポートを $\gamma \cdot \delta$ だけ変動させる．

\paragraph{結果}
\Cref{tab:sutva} に結果を示す．
$\gamma \leq 0.05$（波及効果が真の効果の5\%以下）では MAE の増加は軽微である（0.018 $\to$ 0.023）．
$\gamma = 0.10$ で MAE が 0.039 に増加し，$\gamma = 0.15$ では 0.058 となる．
波及効果が真の効果の10\%を超えると推定精度が顕著に悪化するため，アイテム非共有基準の重要性が確認される．
