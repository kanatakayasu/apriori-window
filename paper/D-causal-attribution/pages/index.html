From Association to Causation: Synthetic Control for Dense Pattern Attribution

SC-DenseAttrib applies the Synthetic Control Method to pattern support time series, enabling causal attribution of support changes to external events.

Target: KDD 2027 / AAAI 2027

Key Concepts

Donor Pool

Item-disjoint patterns serve as control units. Patterns sharing no items with the treated pattern are unaffected by the same intervention.

Counterfactual Trajectory

A weighted combination of donor support series estimates what the treated pattern's support would have been without the intervention.

Causal Effect

The difference between observed and counterfactual support quantifies the intervention's causal impact on the pattern.

Placebo Inference

Each donor is iteratively treated as the "treated unit" to construct a null distribution of effect sizes, yielding exact p-values.

Method Overview

SC-DenseAttrib operates in five stages:

  1. Support Series Computation: Sliding-window support for all patterns
  2. Donor Pool Construction: Select item-disjoint patterns as controls
  3. Weight Estimation: Constrained optimization on pre-intervention fit
  4. Causal Effect Estimation: Observed minus counterfactual support
  5. Placebo Inference: Iterative placebo tests for p-values

Theoretical Guarantees

Theorem 1 (Unbiasedness): Under parallel trends, the SCM estimator is unbiased for the true causal effect.
Theorem 2 (Placebo Validity): Under the null, placebo tests yield exact p-values of 1/(J+1), where J is the donor pool size.
