\section{結論}\label{sec:conclusion}

本研究では、スライディングウィンドウに基づく密集区間マイニングに$(\varepsilon, \delta)$-差分プライバシー保証を付与するDP-DIMフレームワークを提案した。

主要な理論的貢献として、(1) ウィンドウカウントクエリの窓感度が$\Delta_W = 1$であることの証明、(2) ラプラスメカニズムおよびガウスメカニズムに基づくDP密集区間検出アルゴリズム、(3) 閾値安定性条件の導出、(4) Sparse Vector Technique による予算効率的な変種、(5) 逐次合成および高度合成定理に基づくApriori型多レベル探索の予算管理手法を示した。

実験結果より、以下が確認された：
\begin{itemize}
  \item $\varepsilon = 1.0$の実用的なプライバシーレベルで平均F1スコア0.60を達成し、密集パターンの概形を捕捉可能であること。
  \item ラプラスメカニズムがガウスメカニズムおよびSVTと比較して最も高い精度を示すこと。
  \item 実行時間がトランザクション数に対してほぼ線形にスケールし、$n = 2000$でも51 msであること。
  \item 高度合成定理により$k = 500$クエリで逐次合成比77.5\%の予算削減が可能であること。
\end{itemize}

\subsection{今後の課題}

今後の研究方向として以下が挙げられる：

\textbf{局所差分プライバシー}：本研究は中央集権型DPを想定しているが、各ユーザーが自身のトランザクションにローカルにノイズを加える局所差分プライバシー（LDP）モデルへの拡張が求められる。LDPでは感度が異なるため、精度とプライバシーのトレードオフが変化する。

\textbf{実データでの評価}：合成データに加え、小売・医療・ウェブログなどの実データでの評価が必要である。特に、密集パターンの疎密度やアイテム数がDP精度に与える影響を調査する。

\textbf{Renyi差分プライバシー}：RDP~\cite{mironov2017renyi}を用いた合成定理は高度合成よりも厳密な上界を与える場合があり、予算管理のさらなる改善が期待される。

\textbf{適応的予算配分}：候補アイテムセットの数やウィンドウ位置に応じた適応的な予算配分戦略の開発により、固定配分を上回る精度が達成できる可能性がある。
