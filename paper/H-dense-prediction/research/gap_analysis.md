# Paper H: Dense Interval Prediction — Gap Analysis

## Research Question
密集区間の発生パターンを点過程・生存分析の枠組みでモデル化し、
次回の密集区間の発生時刻と持続時間を予測できるか？

## 既存手法との位置づけ

| 手法カテゴリ | 代表手法 | 限界 |
|---|---|---|
| 頻出パターンマイニング | Apriori, FP-Growth, ECLAT | 記述的。将来の発生を予測しない |
| 時系列予測 | Prophet, N-BEATS, ARIMA | 連続値予測。離散的な「区間の発生」を扱えない |
| 点過程 | Hawkes Process | イベント発生を扱うが、パターンマイニングと統合されていない |
| 生存分析 | Cox PH, Kaplan-Meier | 再発イベントへの拡張が必要 |
| バースト検出 | Kleinberg model | 検出のみで予測を行わない |

## 研究ギャップ
1. **記述→予測の橋渡し**: 密集区間マイニングの出力を予測モデルの入力にする枠組みが存在しない
2. **密集区間間時間の分布特性**: Inter-Dense Interval Time (IDIT) の統計的性質が未解明
3. **持続時間予測**: 密集区間の持続時間を予測する手法が存在しない
4. **自己励起性**: 密集区間の発生が後続の密集区間発生を促進する自己励起効果の定量化

## 提案する貢献
1. Dense Interval Occurrence Process (DIOP) の定式化
2. IDIT の分布フィッティングと適合度検定
3. Hawkes-Dense モデルによる次回発生時刻予測
4. 生存分析に基づく持続時間予測
5. 合成データ + 実データでの実験的検証
