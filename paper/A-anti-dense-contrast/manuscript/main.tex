\title{反密集区間とコントラスト密集パターン：\\
密集区間マイニングの対称的拡張}

\author{
\IEEEauthorblockN{Anonymous Author(s)}
}

\maketitle

\begin{abstract}
密集区間マイニングは，パターンのスライディングウィンドウサポートが閾値を超える連続的な時間区間を特定する手法であり，
トランザクションデータにおける持続的な高活性期間を効果的に捉える．
しかし，既存のフレームワークは非対称であり，パターンの\emph{存在}（高サポート）を特徴づけるのみで，
パターンの\emph{不在}（低サポート）や時間レジーム間の構造比較を行う機構を備えていない．
本論文では，二つの対称的拡張を提案する．
第一に，サポートが低閾値を下回り続ける極大連続区間として\emph{反密集区間}を定義し，
効率的なマイニングを可能にする逆単調性を証明する．
第二に，二つの時間レジーム間で密集区間構造を比較するフレームワークとして\emph{コントラスト密集パターン}を導入する．
四種類のトポロジー変化（出現・消失・増幅・収縮）の完全な分類体系を定義し，
偽発見率制御を伴う置換検定に基づく統計的検定を提供する．
両拡張は標準的な密集区間検出と同じ$O(n \log n)$の計算量を有する．
合成データを用いた実験により，反密集区間の復元においてF1 = 0.789，
トポロジー変化分類において76\%の精度，
キャンペーン誘発パターン消失の統計的に有意な検出（$p = 0.005$）を示す．
本フレームワークは，パターン消失期間の特定や構造変化の外部介入への帰属など，
新たな分析能力を実現する．
\end{abstract}
